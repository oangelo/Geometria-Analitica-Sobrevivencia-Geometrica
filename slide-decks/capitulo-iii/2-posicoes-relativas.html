Posições Relativas: Por Que Importa?

No planejamento tático, saber se duas rotas se cruzam é questão de segurança. A rota da patrulha atravessa a zona contaminada? Duas expedições vão se encontrar em campo aberto? Conhecer a posição relativa entre duas retas pode evitar emboscadas — ou planejar encontros estratégicos.

Já sabemos criar equações de retas (Seção 1). Agora, com duas retas no plano, surge a pergunta fundamental: como elas se relacionam geometricamente?

DICA: Identificar a posição relativa de duas retas é o primeiro passo para prever interseções, calcular ângulos e garantir alinhamentos seguros no terreno.

Os Três Casos Possíveis

Dadas duas retas \(r_1\) e \(r_2\) no plano, existem exatamente três possibilidades:

Concorrentes
Exatamente um ponto em comum.
Paralelas
Nenhum ponto em comum.
Coincidentes
Todos os pontos em comum.

Sistemas Lineares 2×2 e Posições

Dadas duas retas na forma geral \(r_1: A_1x + B_1y + C_1 = 0\) e \(r_2: A_2x + B_2y + C_2 = 0\), a posição relativa é determinada pelo sistema linear formado por suas equações — conexão direta com o Capítulo 0.

Sistema: \( \begin{cases} A_1x + B_1y = -C_1 \\ A_2x + B_2y = -C_2 \end{cases} \)

Determinante principal: \( D = A_1B_2 - A_2B_1 \)

  • \( D \neq 0 \) — SPD (Sistema Possível Determinado) ⇔ Retas concorrentes. Solução única = ponto de interseção.
  • \( D = 0 \) — SI (Sistema Impossível) ⇔ Retas paralelas distintas. Nenhuma solução.
  • \( D = 0 \) — SPI (Sistema Possível Indeterminado) ⇔ Retas coincidentes. Infinitas soluções.

A natureza do sistema espelha a geometria. Resolver o sistema é encontrar a interseção!

Classificação Rápida sem Resolver

Não precisamos resolver o sistema inteiro para classificar a posição relativa. Basta comparar as razões dos coeficientes:

Teste das Razões

Se \(\frac{A_1}{A_2} = \frac{B_1}{B_2}\): retas são paralelas ou coincidentes.

Nesse caso, verifique \(\frac{C_1}{C_2}\):

  • Se \(\frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}\) → coincidentes.
  • Se \(\frac{A_1}{A_2} = \frac{B_1}{B_2} \neq \frac{C_1}{C_2}\) → paralelas.

Caso contrário (\(\frac{A_1}{A_2} \neq \frac{B_1}{B_2}\)) → concorrentes.

DICA: Este teste é equivalente a verificar o determinante \(D = A_1B_2 - A_2B_1\). Se \(D \neq 0\), as retas são concorrentes — não precisa calcular razão nenhuma!

Distância entre Retas Paralelas

Se \(r_1 \parallel r_2\), a distância entre elas é constante em qualquer ponto. O método: escolha um ponto \(P\) qualquer sobre \(r_1\) e calcule a distância de \(P\) a \(r_2\).

Fórmula (coeficientes proporcionais)

Se \(r_1: Ax + By + C_1 = 0\) e \(r_2: Ax + By + C_2 = 0\) (mesmos \(A\) e \(B\)):

\( d = \frac{|C_2 - C_1|}{\sqrt{A^2 + B^2}} \)

Exemplo: \(r_1: 3x + 4y - 12 = 0\) e \(r_2: 3x + 4y + 8 = 0\).

\( d = \frac{|8 - (-12)|}{\sqrt{9 + 16}} = \frac{20}{5} = 4 \) unidades.

DICA: Se os coeficientes são apenas proporcionais (não iguais), normalize primeiro. Por exemplo, se \(r_2: 6x + 8y + 16 = 0\), divida por 2 para obter \(3x + 4y + 8 = 0\) antes de aplicar a fórmula.

Síntese: Posições Relativas

Três posições, uma ferramenta central: o sistema linear 2×2.

  • Concorrentes (\(D \neq 0\)): interseção única — resolvemos o sistema.
  • Paralelas (\(D = 0\), SI): sem interseção — distância constante.
  • Coincidentes (\(D = 0\), SPI): mesma reta — infinitos pontos comuns.

Sabemos identificar se retas são paralelas. Mas e se precisarmos construir uma reta paralela ou perpendicular a uma reta dada? Essa é a missão da Seção 3.

Antes de seguir, pratique a classificação rápida no simulador. A intuição geométrica é tão importante quanto a técnica algébrica.
