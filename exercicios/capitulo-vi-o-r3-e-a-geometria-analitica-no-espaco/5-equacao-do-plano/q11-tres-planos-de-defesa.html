Três Planos de Defesa: Interseção no Espaço

Objetivo: Conectar a equação do plano com sistemas lineares 3×3 (Tópico 7), encontrando a interseção de três planos.

Três planos de defesa automatizada foram posicionados no espaço:

$\pi_1: x + y + z = 6$

$\pi_2: 2x - y + z = 3$

$\pi_3: x + 2y - z = 2$

(a) Resolva o sistema linear 3×3 para encontrar o ponto de interseção dos três planos.

(b) Verifique que o ponto encontrado satisfaz as três equações.

(c) Os vetores normais dos três planos são linearmente independentes? O que isso significa geometricamente?

DICA DE SOBREVIVÊNCIA
Resolva o sistema por eliminação. Some $\pi_1$ e $\pi_3$ para eliminar $z$. Subtraia $\pi_1$ de $\pi_2$ (ou manipule coeficientes) para obter outra equação sem $z$. Resolva o sistema 2×2 resultante para $x$ e $y$, depois substitua para encontrar $z$. Para (c), calcule o determinante da matriz formada pelos três vetores normais.
ACESSO NÍVEL: SUPERVISOR
  1. Item (a) — Sistema: $$\begin{cases} x + y + z = 6 \quad (\pi_1) \\ 2x - y + z = 3 \quad (\pi_2) \\ x + 2y - z = 2 \quad (\pi_3) \end{cases}$$
  2. Somar $\pi_1 + \pi_3$: $(x + y + z) + (x + 2y - z) = 6 + 2 \Rightarrow 2x + 3y = 8$ (Eq. 4).
  3. Subtrair $\pi_2 - \pi_1$: $(2x - y + z) - (x + y + z) = 3 - 6 \Rightarrow x - 2y = -3$ (Eq. 5).
  4. De Eq. 5: $x = 2y - 3$. Substituir em Eq. 4: $2(2y - 3) + 3y = 8 \Rightarrow 4y - 6 + 3y = 8 \Rightarrow 7y = 14 \Rightarrow y = 2$.
  5. $x = 2(2) - 3 = 1$.
  6. De $\pi_1$: $1 + 2 + z = 6 \Rightarrow z = 3$.
  7. Item (b) — Verificação: $\pi_1: 1 + 2 + 3 = 6$ ✓; $\pi_2: 2(1) - 2 + 3 = 3$ ✓; $\pi_3: 1 + 2(2) - 3 = 2$ ✓.
  8. Item (c) — Vetores normais: $\vec{n}_1 = (1, 1, 1)$, $\vec{n}_2 = (2, -1, 1)$, $\vec{n}_3 = (1, 2, -1)$. Determinante: $$\det = \begin{vmatrix} 1 & 1 & 1 \\ 2 & -1 & 1 \\ 1 & 2 & -1 \end{vmatrix} = 1(1 - 2) - 1(-2 - 1) + 1(4 + 1) = -1 + 3 + 5 = 7 \neq 0$$ Os vetores são linearmente independentes, logo os três planos se intersectam em um único ponto.
$$P(1, 2, 3) \qquad \det(\vec{n}_1, \vec{n}_2, \vec{n}_3) = 7 \neq 0 \Rightarrow \text{interseção única}$$

Reflexão de Sobrevivência: Três planos com normais linearmente independentes se cruzam em exatamente um ponto — como três paredes de um canto de sala. Se os normais fossem dependentes, os planos poderiam ser paralelos (sem interseção) ou concorrentes em uma reta (infinitos pontos de interseção). Este é o fundamento geométrico da teoria de sistemas lineares.
