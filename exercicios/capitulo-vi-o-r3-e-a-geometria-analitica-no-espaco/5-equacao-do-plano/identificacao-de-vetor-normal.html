Identificação de Vetor Normal

Os engenheiros do Refúgio-Tec encontraram os planos de uma antiga usina de energia. Uma das paredes principais tem equação $2x - 3y + z - 6 = 0$. Para instalar corretamente os painéis de blindagem, precisam identificar o vetor normal a esta parede.

DICA DE SOBREVIVÊNCIA
Na equação geral $ax + by + cz + d = 0$, o vetor normal é simplesmente $(a, b, c)$.
ACESSO NÍVEL: SUPERVISOR
  1. Identificar a equação na forma geral: $2x - 3y + z - 6 = 0$
  2. Extrair os coeficientes das variáveis: $a = 2$, $b = -3$, $c = 1$
  3. Formar o vetor normal: $\vec{n} = (2, -3, 1)$
$$\vec{n} = (2, -3, 1)$$

Reflexão de Sobrevivência: Conhecer a direção normal de uma parede é essencial para calcular ângulos de incidência de projéteis ou radiação solar, otimizando tanto a defesa quanto o aproveitamento energético do abrigo.
