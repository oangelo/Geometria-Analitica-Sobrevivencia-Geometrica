DECOMPOSIÇÃO EM VETORES CANÔNICOS

O objetivo deste exercício é decompor um vetor do R³ como combinação linear dos vetores canônicos $\vec{i}$, $\vec{j}$ e $\vec{k}$. Esta decomposição é a base para operações mais avançadas que veremos nos próximos tópicos, como o produto vetorial.

Um drone de reconhecimento recebe instruções de deslocamento $\vec{v} = (5, -2, 3)$ para se reposicionar após uma tempestade de areia.

(a) Escreva $\vec{v}$ na forma $\vec{v} = v_1\vec{i} + v_2\vec{j} + v_3\vec{k}$.

(b) Calcule $2\vec{v} - 3\vec{i} + 4\vec{j} - \vec{k}$ usando a decomposição canônica.

(c) Mostre que $\vec{i}$, $\vec{j}$ e $\vec{k}$ são vetores unitários e mutuamente perpendiculares (dois a dois).

DICA DE SOBREVIVÊNCIA

Os vetores canônicos são $\vec{i} = (1, 0, 0)$, $\vec{j} = (0, 1, 0)$, $\vec{k} = (0, 0, 1)$. A decomposição é direta: cada componente de $\vec{v}$ multiplica o canônico correspondente. Para o item (c), calcule módulos e observe que vetores perpendiculares têm produto escalar nulo (conceito do Cap I estendido ao R³).

ACESSO NÍVEL: SUPERVISOR
  1. Dado: $\vec{v} = (5, -2, 3)$
  2. Item (a) — $\vec{v} = 5\vec{i} + (-2)\vec{j} + 3\vec{k} = 5\vec{i} - 2\vec{j} + 3\vec{k}$
  3. Item (b) — $2\vec{v} = 10\vec{i} - 4\vec{j} + 6\vec{k}$
  4. $2\vec{v} - 3\vec{i} + 4\vec{j} - \vec{k} = (10 - 3)\vec{i} + (-4 + 4)\vec{j} + (6 - 1)\vec{k} = 7\vec{i} + 0\vec{j} + 5\vec{k}$
  5. Resultado: $(7, 0, 5)$
  6. Item (c) — Módulos: $|\vec{i}| = \sqrt{1^2 + 0^2 + 0^2} = 1$, $|\vec{j}| = 1$, $|\vec{k}| = 1$ ✓
  7. Produtos escalares: $\vec{i} \cdot \vec{j} = 1 \cdot 0 + 0 \cdot 1 + 0 \cdot 0 = 0$ ⟹ perpendiculares
  8. $\vec{i} \cdot \vec{k} = 0$ e $\vec{j} \cdot \vec{k} = 0$ ✓
$$\vec{v} = 5\vec{i} - 2\vec{j} + 3\vec{k}$$ $$2\vec{v} - 3\vec{i} + 4\vec{j} - \vec{k} = 7\vec{i} + 5\vec{k} = (7, 0, 5)$$

Reflexão de Sobrevivência: Os vetores canônicos formam uma base ortonormal do R³: são unitários, perpendiculares dois a dois, e qualquer vetor se escreve unicamente como combinação linear deles. Esta propriedade é o alicerce de toda a geometria analítica no espaço. Quando você decompõe $\vec{v} = 5\vec{i} - 2\vec{j} + 3\vec{k}$, está traduzindo a linguagem geométrica para a linguagem algébrica.
