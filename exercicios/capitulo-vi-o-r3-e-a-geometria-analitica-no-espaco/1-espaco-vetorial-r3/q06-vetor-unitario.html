VETOR UNITÁRIO NO R³

O objetivo deste exercício é construir um vetor unitário (de módulo 1) na mesma direção de um vetor dado. Esta operação — chamada de normalização — é essencial para descrever direções sem se preocupar com magnitude, como quando um drone precisa seguir uma rota independentemente de sua velocidade.

Uma sonda de reconhecimento deve ser lançada na direção do vetor $\vec{v} = (2, -4, 4)$, mas com velocidade unitária para economia de combustível.

(a) Calcule $|\vec{v}|$.

(b) Determine o vetor unitário $\vec{u} = \dfrac{\vec{v}}{|\vec{v}|}$.

(c) Verifique que $|\vec{u}| = 1$.

(d) O que acontece com $\vec{u}$ se multiplicarmos $\vec{v}$ por $-3$ antes de normalizar? Explique.

DICA DE SOBREVIVÊNCIA

Um vetor unitário na direção de $\vec{v}$ é obtido dividindo cada componente pelo módulo de $\vec{v}$. A verificação do item (c) confirma que a normalização funcionou. Para o item (d), lembre-se de que multiplicar por um escalar negativo inverte a direção.

ACESSO NÍVEL: SUPERVISOR
  1. Dado: $\vec{v} = (2, -4, 4)$
  2. Item (a) — $|\vec{v}| = \sqrt{2^2 + (-4)^2 + 4^2} = \sqrt{4 + 16 + 16} = \sqrt{36} = 6$
  3. Item (b) — $\vec{u} = \dfrac{\vec{v}}{|\vec{v}|} = \left(\dfrac{2}{6}, \; \dfrac{-4}{6}, \; \dfrac{4}{6}\right) = \left(\dfrac{1}{3}, \; -\dfrac{2}{3}, \; \dfrac{2}{3}\right)$
  4. Item (c) — $|\vec{u}| = \sqrt{\left(\dfrac{1}{3}\right)^2 + \left(-\dfrac{2}{3}\right)^2 + \left(\dfrac{2}{3}\right)^2} = \sqrt{\dfrac{1 + 4 + 4}{9}} = \sqrt{\dfrac{9}{9}} = 1$ ✓
  5. Item (d) — Seja $\vec{w} = -3\vec{v} = (-6, 12, -12)$
  6. $|\vec{w}| = |-3| \cdot |\vec{v}| = 3 \cdot 6 = 18$
  7. Vetor unitário de $\vec{w}$: $\dfrac{\vec{w}}{|\vec{w}|} = \dfrac{-3\vec{v}}{3|\vec{v}|} = -\dfrac{\vec{v}}{|\vec{v}|} = -\vec{u} = \left(-\dfrac{1}{3}, \; \dfrac{2}{3}, \; -\dfrac{2}{3}\right)$
$$\vec{u} = \left(\frac{1}{3}, -\frac{2}{3}, \frac{2}{3}\right), \quad |\vec{u}| = 1$$ $$\text{Normalização de } (-3\vec{v}) \text{ produz } -\vec{u}$$

Reflexão de Sobrevivência: O vetor unitário captura apenas a direção, descartando a magnitude. Multiplicar por escalar negativo antes de normalizar inverte a direção do unitário resultante. Isso é intuitivo: se a sonda deve ir para o lado oposto, o vetor unitário aponta para o lado oposto.
