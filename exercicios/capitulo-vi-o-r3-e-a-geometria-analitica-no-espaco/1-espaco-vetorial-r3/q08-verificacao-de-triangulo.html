VERIFICAÇÃO DE TRIÂNGULO NO ESPAÇO

O objetivo deste exercício é usar vetores para verificar se três pontos no espaço formam um triângulo válido — ou seja, se são não colineares. Três pontos são colineares quando os vetores que os conectam são múltiplos escalares um do outro.

Três sondas de reconhecimento reportaram posições de possíveis abrigos: $A(1, 2, 3)$, $B(4, 5, 6)$ e $C(7, 8, 9)$. Antes de despachar equipes, você precisa confirmar que estas posições definem uma área de busca triangular (não uma linha reta).

(a) Calcule os vetores $\overrightarrow{AB}$ e $\overrightarrow{AC}$.

(b) Verifique se existe um escalar $k$ tal que $\overrightarrow{AC} = k\overrightarrow{AB}$. O que isso implica sobre a colinearidade?

(c) Se os pontos forem colineares, proponha uma pequena alteração na coordenada $z$ de $C$ que torne os três pontos não colineares. Justifique.

(d) Esboce a configuração dos pontos em perspectiva isométrica, indicando se estão alinhados ou formam um triângulo.

DICA DE SOBREVIVÊNCIA

Dois vetores são paralelos (e portanto os pontos são colineares) se um é múltiplo escalar do outro. Verifique componente a componente: se $\overrightarrow{AC} = k\overrightarrow{AB}$, então cada componente de $\overrightarrow{AC}$ deve ser $k$ vezes a correspondente componente de $\overrightarrow{AB}$. Para o item (c), alterar uma coordenada quebrará a proporção.

ACESSO NÍVEL: SUPERVISOR
  1. Item (a) — $\overrightarrow{AB} = (4 - 1, \; 5 - 2, \; 6 - 3) = (3, 3, 3)$
  2. $\overrightarrow{AC} = (7 - 1, \; 8 - 2, \; 9 - 3) = (6, 6, 6)$
  3. Item (b) — Verificando: $\dfrac{6}{3} = 2$, $\dfrac{6}{3} = 2$, $\dfrac{6}{3} = 2$
  4. Portanto, $\overrightarrow{AC} = 2\overrightarrow{AB}$, com $k = 2$
  5. Os pontos $A$, $B$ e $C$ são COLINEARES — não formam triângulo
  6. Item (c) — Se alterarmos $C$ para $C' = (7, 8, 10)$, então $\overrightarrow{AC'} = (6, 6, 7)$
  7. Verificando: $\dfrac{6}{3} = 2$, $\dfrac{6}{3} = 2$, mas $\dfrac{7}{3} \neq 2$
  8. Não existe $k$ tal que $\overrightarrow{AC'} = k\overrightarrow{AB}$ ⟹ $A$, $B$, $C'$ são não colineares ✓
$$\overrightarrow{AB} = (3, 3, 3), \quad \overrightarrow{AC} = (6, 6, 6)$$ $$\overrightarrow{AC} = 2\overrightarrow{AB} \Rightarrow A, B, C \text{ são colineares}$$ $$C' = (7, 8, 10) \Rightarrow \overrightarrow{AC'} = (6, 6, 7) \text{ (não colinear)}$$

Reflexão de Sobrevivência: A colinearidade dos pontos $A$, $B$, $C$ é um alerta: as três sondas estavam na mesma linha de visada. Isso pode indicar que seguiram a mesma rota ou que há um erro de calibração. Na Zona Devastada, três pontos colineares oferecem cobertura de busca mínima — um triângulo real expande a área de reconhecimento significativamente.
