Interpretação Geométrica dos Planos

Objetivo: Associar a descrição geométrica de três planos à classificação do sistema correspondente.

Para cada configuração geométrica abaixo, esboce mentalmente (ou descreva por escrito) a disposição dos planos e determine se o sistema resultante é SPD, SPI ou SI.

a) Três planos se intersectam dois a dois em retas distintas, e as três retas de interseção se encontram num único ponto.

b) Dois planos são paralelos distintos; o terceiro plano intersecta ambos.

c) Os três planos contêm uma mesma reta comum.

d) Os três planos são coincidentes (a mesma equação multiplicada por constantes não nulas).

e) Cada par de planos se intersecta numa reta, mas as três retas de interseção são paralelas distintas (configuração de prisma triangular).

f) Dois planos são coincidentes; o terceiro os intersecta numa reta.

g) Para o item (a), escreva um exemplo numérico de sistema 3×3 que tenha essa configuração (dica: use o sistema do Q1).

DICA DE SOBREVIVÊNCIA

Pense no que acontece com o número de soluções em cada caso. SPD = 1 ponto, SPI = infinitos pontos (reta ou plano), SI = nenhum ponto. Para o item (g), lembre-se de que três planos em posição geral (normais linearmente independentes) se intersectam num ponto.

ACESSO NÍVEL: SUPERVISOR

a) SPD. Os três planos se encontram num único ponto. É a configuração genérica: os vetores normais são linearmente independentes.

b) SI. Dois planos paralelos distintos nunca se intersectam, logo não há ponto comum aos três. O terceiro plano intersecta cada um dos paralelos em retas paralelas distintas.

c) SPI. Infinitas soluções: todos os pontos da reta comum. Os três planos são distintos mas compartilham uma reta.

d) SPI. Todos os pontos do plano são solução. Geometricamente, os três planos são o mesmo plano. O sistema tem duas variáveis livres.

e) SI. Cada par de planos se intersecta, mas não há ponto comum aos três. As três retas de interseção são paralelas distintas, como as arestas de um prisma triangular.

f) SPI. Dois planos coincidentes formam um único plano; o terceiro o intersecta numa reta. Infinitas soluções (pontos dessa reta).

g) Exemplo (sistema do Q1):

$$\begin{cases} x + y + z = 6 \\ 2x - y + z = 3 \\ x + 2y - z = 2 \end{cases}$$

Vetores normais: $\vec{n}_1 = (1,1,1)$, $\vec{n}_2 = (2,-1,1)$, $\vec{n}_3 = (1,2,-1)$. Como $\det A = -9 \neq 0$, os vetores são linearmente independentes e os planos se intersectam num único ponto: $P(1, 2, 3)$.
