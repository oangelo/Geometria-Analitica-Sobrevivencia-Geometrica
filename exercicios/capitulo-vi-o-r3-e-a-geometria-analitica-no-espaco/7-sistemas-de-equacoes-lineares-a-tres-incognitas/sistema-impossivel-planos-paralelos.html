Sistema Impossível: Planos Paralelos

Objetivo: Reconhecer um sistema impossível (SI) e identificar a contradição que surge durante o escalonamento.

Três barreiras de contenção devem ser posicionadas segundo equações que definem planos no espaço. O engenheiro responsável detectou inconsistência nos dados:

$$\begin{cases} x + y + z = 4 \\ 2x + 2y + 2z = 10 \\ x - y + 2z = 1 \end{cases}$$

a) Observe as equações (I) e (II). Os planos correspondentes são paralelos? Justifique comparando os vetores normais.

b) Aplique o método de eliminação: elimine $x$ das equações (II) e (III) usando a (I) como pivô.

c) O que acontece quando você tenta resolver o sistema 2×2 resultante? Identifique a contradição.

d) Classifique o sistema (SPD, SPI ou SI) e interprete geometricamente a configuração dos três planos.

e) Para tornar o sistema possível, qual deveria ser o termo independente da equação (II)? Justifique.

DICA DE SOBREVIVÊNCIA

Dois planos são paralelos quando seus vetores normais são paralelos (múltiplos um do outro) mas os planos são distintos (termos independentes diferentes quando normalizados). A contradição aparece como uma equação do tipo $0 = k$ com $k \neq 0$.

ACESSO NÍVEL: SUPERVISOR

a) Vetor normal do primeiro plano: $\vec{n}_1 = (1, 1, 1)$. Vetor normal do segundo plano: $\vec{n}_2 = (2, 2, 2) = 2(1, 1, 1) = 2\vec{n}_1$.

Os vetores normais são paralelos, logo os planos são paralelos ou coincidentes. Para verificar se são coincidentes, normalizamos: a equação (II) dividida por 2 daria $x + y + z = 5$, que difere do lado direito da (I) ($4 \neq 5$). Portanto, os planos são paralelos distintos.

b) Escalonamento:

Eq. (I) × $(-2)$ + Eq. (II): $(2x + 2y + 2z) + (-2x - 2y - 2z) = 10 + (-8)$

$$0 = 2 \quad \text{(II')}$$

Eq. (I) × $(-1)$ + Eq. (III): $(x - y + 2z) + (-x - y - z) = 1 + (-4)$

$$-2y + z = -3 \quad \text{(III')}$$

c) A equação (II') é $0 = 2$, uma contradição. Não existe valor de $x$, $y$ ou $z$ que satisfaça $0 = 2$. O sistema não tem solução.

d) O sistema é SI (Sistema Impossível). Geometricamente, os planos (I) e (II) são paralelos distintos, logo nunca se intersectam. O terceiro plano intersecta ambos em retas paralelas distintas, formando uma configuração de prisma triangular (as três retas de interseção dois a dois são paralelas e distintas).

e) Para que os planos (I) e (II) fossem coincidentes (e o sistema passível de ter solução), o termo independente da (II) deveria ser $8$, pois $2x + 2y + 2z = 8$ equivale a $x + y + z = 4$, idêntico à equação (I). Nesse caso, o sistema teria infinitas soluções (SPI).
