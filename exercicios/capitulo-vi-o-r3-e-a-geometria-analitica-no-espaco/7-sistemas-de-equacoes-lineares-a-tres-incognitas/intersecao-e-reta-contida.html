Interseção e Reta Contida

Objetivo: Sintetizar todos os conceitos do tópico: resolver sistema 3×3, encontrar reta de interseção, e verificar condição de contenção.

Dados os três planos:

$$\begin{cases} \pi_1: x + y + z = 1 \\ \pi_2: 2x - y + z = 2 \\ \pi_3: x + 2y - z = 3 \end{cases}$$

a) Calcule $\det A$ para a matriz de coeficientes. O sistema é SPD, SPI ou SI?

b) Resolva o sistema por eliminação, encontrando o ponto $P$ de interseção dos três planos (se existir).

c) Encontre a reta $r = \pi_1 \cap \pi_2$, ou seja, a interseção dos dois primeiros planos. Escreva as equações paramétricas de $r$.

d) Verifique se a reta $r$ está contida no plano $\pi_3$. Para isso, verifique se dois pontos distintos de $r$ pertencem a $\pi_3$.

e) Se a reta $r$ não estiver contida em $\pi_3$, encontre o ponto de interseção $r \cap \pi_3$. Compare com o ponto $P$ encontrado no item (b).

f) Considere agora o plano $\pi_4: 3x + z = 3$. Verifique se $\pi_4$ contém a reta $r$. Se não, encontre $r \cap \pi_4$.

g) Desafio: Determine a equação geral de todos os planos que contêm a reta $r = \pi_1 \cap \pi_2$. Dica: use combinação linear das equações de $\pi_1$ e $\pi_2$.

DICA DE SOBREVIVÊNCIA

Para verificar se uma reta está contida num plano, basta verificar se dois pontos distintos da reta satisfazem a equação do plano. Se um plano contém uma reta, então todo ponto da reta satisfaz a equação do plano. A família de planos por uma reta é dada por $\alpha \cdot \pi_1 + \beta \cdot \pi_2 = 0$ (combinação linear das equações).

ACESSO NÍVEL: SUPERVISOR

a) Matriz:

$$A = \begin{pmatrix} 1 & 1 & 1 \\ 2 & -1 & 1 \\ 1 & 2 & -1 \end{pmatrix}$$

$\det A = 1(-1)(-1) + 1(1)(1) + 1(2)(2) - 1(-1)(1) - 1(1)(2) - 1(2)(-1)$ $= 1 + 1 + 4 + 1 - 2 + 2 = 7 \neq 0$.

O sistema é SPD — solução única.

b) Escalonamento:

Eq. (I) × $(-2)$ + Eq. (II): $-3y - z = 0$ (II')

Eq. (I) × $(-1)$ + Eq. (III): $y - 2z = 2$ (III')

De (III'): $y = 2z + 2$. Substituindo em (II'): $-3(2z+2) - z = 0 \Rightarrow -6z - 6 - z = 0 \Rightarrow -7z = 6 \Rightarrow z = -\frac{6}{7}$.

Logo $y = 2(-\frac{6}{7}) + 2 = -\frac{12}{7} + \frac{14}{7} = \frac{2}{7}$.

Da equação (I): $x = 1 - y - z = 1 - \frac{2}{7} + \frac{6}{7} = \frac{7 - 2 + 6}{7} = \frac{11}{7}$.

Solução: $P\left(\dfrac{11}{7}, \dfrac{2}{7}, -\dfrac{6}{7}\right)$.

c) Reta $r = \pi_1 \cap \pi_2$:

$$\begin{cases} x + y + z = 1 \\ 2x - y + z = 2 \end{cases}$$

Subtraindo (I) de (II): $x - 2y = 1 \Rightarrow x = 2y + 1$.

Substituindo em (I): $(2y+1) + y + z = 1 \Rightarrow z = -3y$.

Fazendo $y = t$:

$$r: \begin{cases} x = 1 + 2t \\ y = t \\ z = -3t \end{cases}, \quad t \in \mathbb{R}$$

d) Verificando se $r \subset \pi_3$:

Tomando dois pontos de $r$:

  • $t = 0$: $A(1, 0, 0)$. Em $\pi_3$: $1 + 2(0) - 0 = 1 \neq 3$. $A \notin \pi_3$.

Como $A \notin \pi_3$, a reta $r$ não está contida em $\pi_3$.

e) Interseção $r \cap \pi_3$: substituímos as equações paramétricas de $r$ em $\pi_3$:

$$(1 + 2t) + 2(t) - (-3t) = 3 \Rightarrow 1 + 2t + 2t + 3t = 3 \Rightarrow 7t = 2 \Rightarrow t = \frac{2}{7}$$

Logo: $x = 1 + \frac{4}{7} = \frac{11}{7}$, $y = \frac{2}{7}$, $z = -\frac{6}{7}$.

O ponto de interseção é $\left(\frac{11}{7}, \frac{2}{7}, -\frac{6}{7}\right) = P$, coincidente com a solução do sistema. Isso confirma que $P$ é o único ponto comum aos três planos.

f) Verificando $\pi_4: 3x + z = 3$:

Substituindo $r$: $3(1+2t) + (-3t) = 3 + 6t - 3t = 3 + 3t = 3 \Rightarrow 3t = 0 \Rightarrow t = 0$.

A reta $r$ não está contida em $\pi_4$ (só intersecta no ponto $t=0$, ou seja, $A(1,0,0)$).

$r \cap \pi_4 = \{(1, 0, 0)\}$.

g) A família de planos que contêm a reta $r = \pi_1 \cap \pi_2$ é dada pela combinação linear:

$$\alpha(x + y + z - 1) + \beta(2x - y + z - 2) = 0$$

onde $\alpha, \beta \in \mathbb{R}$ e não ambos nulos.

Expandindo: $(\alpha + 2\beta)x + (\alpha - \beta)y + (\alpha + \beta)z - (\alpha + 2\beta) = 0$.

Para $\alpha = 1, \beta = 0$: recuperamos $\pi_1$. Para $\alpha = 0, \beta = 1$: recuperamos $\pi_2$. Para $\alpha = 1, \beta = 1$: obtemos $3x + z = 3$, que é $\pi_4$! Isso explica por que $\pi_4$ intersecta $r$ num único ponto: $\pi_4$ pertence à família, mas como é combinação com $\alpha = \beta = 1$, ele contém a reta $r$? Verificação: $3(1+2t) + (-3t) = 3 + 3t$. Para que seja identicamente 3, precisaríamos $3t = 0$ para todo $t$, o que é falso. Correção: $\pi_4$ não contém $r$ completamente, apenas intersecta em $t=0$. A família correta é toda combinação linear não trivial de $\pi_1$ e $\pi_2$.
