Sistema com Infinitas Soluções (SPI)

Objetivo: Identificar e resolver um sistema 3×3 onde uma equação é combinação linear das outras, resultando em infinitas soluções.

Três sensores de radiação estão calibrados de forma que a leitura do terceiro é sempre a soma das leituras dos dois primeiros. As equações de calibração são:

$$\begin{cases} x + 2y - z = 3 \\ 2x - y + z = 1 \\ 3x + y = 4 \end{cases}$$

a) Verifique que a terceira equação é a soma das duas primeiras. O que isso significa geometricamente?

b) Escalone o sistema: elimine $x$ das duas últimas equações usando a primeira como pivô.

c) O que acontece quando você tenta resolver o sistema 2×2 resultante? Explique por que isso indica infinitas soluções.

d) Escolha $z = t$ (parâmetro livre) e expresse $x$ e $y$ em função de $t$. Escreva a solução geral na forma paramétrica.

e) Dê três soluções particulares escolhendo valores específicos para $t$.

f) Classifique o sistema e interprete geometricamente: como os três planos se posicionam no espaço?

DICA DE SOBREVIVÊNCIA

Se a terceira equação for combinação das outras duas, ela não acrescenta informação nova. Geometricamente, o terceiro plano contém a reta de interseção dos dois primeiros. Para a solução paramétrica, isole $x$ e $y$ em função de $z = t$ usando duas equações independentes.

ACESSO NÍVEL: SUPERVISOR

a) Soma das duas primeiras:

$$(x + 2y - z) + (2x - y + z) = 3x + y = 3 + 1 = 4$$

A terceira equação é exatamente $3x + y = 4$. Geometricamente, o terceiro plano contém a reta de interseção dos dois primeiros planos.

b) Escalonamento:

Eq. (I) × $(-2)$ + Eq. (II): $(2x - y + z) + (-2x - 4y + 2z) = 1 + (-6)$

$$-5y + 3z = -5 \quad \text{(II')}$$

Eq. (I) × $(-3)$ + Eq. (III): $(3x + y) + (-3x - 6y + 3z) = 4 + (-9)$

$$-5y + 3z = -5 \quad \text{(III')}$$

Note que (II') = (III').

c) O sistema 2×2 "resultante" é:

$$\begin{cases} -5y + 3z = -5 \\ -5y + 3z = -5 \end{cases}$$

Subtraindo uma da outra: $0 = 0$. Esta identidade indica que as duas equações são equivalentes — há apenas uma equação independente para duas incógnitas, deixando uma variável livre. Logo, existem infinitas soluções.

d) Fazendo $z = t$ na equação (II'): $-5y + 3t = -5 \Rightarrow y = \dfrac{3t + 5}{5} = \dfrac{3}{5}t + 1$.

Da equação (III): $3x + y = 4 \Rightarrow x = \dfrac{4 - y}{3} = \dfrac{4 - (\frac{3}{5}t + 1)}{3} = \dfrac{3 - \frac{3}{5}t}{3} = 1 - \dfrac{1}{5}t$.

Solução geral:

$$\begin{cases} x = 1 - \dfrac{1}{5}t \\ y = 1 + \dfrac{3}{5}t \\ z = t \end{cases}, \quad t \in \mathbb{R}$$

e) Três soluções particulares:

  • $t = 0$: $(1, 1, 0)$
  • $t = 5$: $(0, 4, 5)$
  • $t = -5$: $(2, -2, -5)$

f) O sistema é SPI (Sistema Possível e Indeterminado). Geometricamente, os três planos se intersectam em uma reta comum. Cada ponto dessa reta é uma solução do sistema.
