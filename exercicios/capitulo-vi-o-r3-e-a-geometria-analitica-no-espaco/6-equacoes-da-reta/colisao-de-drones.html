Colisão de Drones — Problema Aberto

Objetivo: Resolver um problema de verificação de colisão entre duas trajetórias retilíneas no espaço, exigindo que dois objetos ocupem o mesmo ponto no mesmo instante.

Dois drones patrulham a Zona Devastada com trajetórias retilíneas:

  • Drone Alfa: parte de $A_0(0, 0, 10)$ com velocidade $\vec{v}_\alpha = (2, 1, -1)$ m/s.
  • Drone Beta: parte de $B_0(6, 3, 8)$ com velocidade $\vec{v}_\beta = (-1, 0, 1)$ m/s.

Ambos partem em $t = 0$.

  1. Escreva as equações paramétricas de ambas as trajetórias.
  2. Verifique se as retas suporte das trajetórias se intersectam (mesmo ponto no espaço, possivelmente em tempos diferentes).
  3. Se as retas se intersectarem, verifique se a colisão ocorre no mesmo instante $t$ para ambos os drones. Se não, calcule os tempos $t_\alpha$ e $t_\beta$ em que cada drone passa pelo ponto de interseção.
  4. As trajetórias são seguras (sem risco de colisão) ou devem ser alteradas?
DICA DE SOBREVIVÊNCIA
Para verificar interseção de retas, iguale as coordenadas: $x_\alpha(t_\alpha) = x_\beta(t_\beta)$, $y_\alpha(t_\alpha) = y_\beta(t_\beta)$, $z_\alpha(t_\alpha) = z_\beta(t_\beta)$. Isso gera um sistema de 3 equações com 2 incógnitas ($t_\alpha$, $t_\beta$). Se o sistema for consistente, as retas se intersectam. Para colisão, exija $t_\alpha = t_\beta = t$.
ACESSO NÍVEL: SUPERVISOR

Item (a) — Equações paramétricas:

$$\text{Alfa: } \begin{cases} x = 2t_\alpha \\ y = t_\alpha \\ z = 10 - t_\alpha \end{cases}, \quad t_\alpha \geq 0$$ $$\text{Beta: } \begin{cases} x = 6 - t_\beta \\ y = 3 \\ z = 8 + t_\beta \end{cases}, \quad t_\beta \geq 0$$

Item (b) — Interseção das retas:

  1. Igualar coordenadas (usando $t$ para Alfa e $s$ para Beta): $$\begin{cases} 2t = 6 - s \quad \text{(x)} \\ t = 3 \quad \text{(y)} \\ 10 - t = 8 + s \quad \text{(z)} \end{cases}$$
  2. Da equação de $y$: $t = 3$.
  3. Substituir $t = 3$ na equação de $x$: $2(3) = 6 - s \Rightarrow 6 = 6 - s \Rightarrow s = 0$.
  4. Verificar na equação de $z$: $10 - 3 = 8 + 0 \Rightarrow 7 = 8$.

✗ Contradição! $7 \neq 8$. As retas não se intersectam.

Item (c) — Análise de colisão:

Como as retas não se intersectam, não há possibilidade de colisão. Os drones nunca ocuparão o mesmo ponto no espaço, independentemente do tempo.

Verificação adicional — Distância mínima entre as retas:

Para confirmar que as trajetórias são seguras, calculemos a distância mínima entre as duas retas. A fórmula para distância entre retas reversas é:

$$d(r_\alpha, r_\beta) = \frac{|(\vec{B_0} - \vec{A_0}) \cdot (\vec{v}_\alpha \times \vec{v}_\beta)|}{|\vec{v}_\alpha \times \vec{v}_\beta|}$$
  1. $\vec{B_0} - \vec{A_0} = (6, 3, 8) - (0, 0, 10) = (6, 3, -2)$.
  2. $\vec{v}_\alpha \times \vec{v}_\beta = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 2 & 1 & -1 \\ -1 & 0 & 1 \end{vmatrix} = \vec{i}(1 - 0) - \vec{j}(2 - 1) + \vec{k}(0 + 1) = (1, -1, 1)$.
  3. $|\vec{v}_\alpha \times \vec{v}_\beta| = \sqrt{1 + 1 + 1} = \sqrt{3}$.
  4. Produto misto: $(6, 3, -2) \cdot (1, -1, 1) = 6 - 3 - 2 = 1$.
  5. Distância mínima: $d = \frac{|1|}{\sqrt{3}} = \frac{\sqrt{3}}{3} \approx 0{,}58$ m.

Item (d) — Conclusão de segurança:

✓ As trajetórias são seguras. As retas são reversas (não se intersectam e não são paralelas). A distância mínima entre elas é $\frac{\sqrt{3}}{3} \approx 0{,}58$ metros — uma distância muito pequena, mas não nula. Em operações reais, drones precisam de uma margem de segurança maior (tipicamente 5-10 m). Recomenda-se ajustar uma das trajetórias para aumentar esta distância.

$$r_\alpha \cap r_\beta = \emptyset \text{ (retas reversas)}$$ $$d_{\text{mín}} = \frac{\sqrt{3}}{3} \approx 0{,}58 \text{ m}$$ $$\text{Colisão: IMPOSSÍVEL (mas margem de segurança insuficiente)}$$

Reflexão de Sobrevivência: Este é o problema mais completo da lista. A verificação de colisão exige duas condições simultâneas: (1) as retas devem se intersectar no espaço, e (2) a interseção deve ocorrer no mesmo instante $t$. Neste caso, as retas nem sequer se intersectam — são retas reversas, uma configuração típica do R³ que não existe no R². A distância mínima de ~0,58 m é teoricamente segura (não há colisão), mas operacionalmente perigosa. Este exercício ilustra por que o espaço tridimensional é mais complexo que o plano: no R², duas retas ou são paralelas ou se intersectam; no R³, há uma terceira possibilidade — retas reversas, que "passam perto" mas nunca se tocam.
