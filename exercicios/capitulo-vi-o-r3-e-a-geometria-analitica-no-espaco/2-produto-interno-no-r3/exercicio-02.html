Módulo e Ângulo no Espaço

Objetivo: Aplicar as fórmulas de módulo e ângulo no R³, verificando que são idênticas às do R², apenas com três componentes.

Dados $\vec{a} = (1, 2, 2)$ e $\vec{b} = (3, 0, 4)$:

  1. Calcule $|\vec{a}|$ e $|\vec{b}|$.
  2. Calcule o ângulo $\theta$ entre $\vec{a}$ e $\vec{b}$ (em graus, aproximado para inteiro).
  3. Esboce mentalmente (ou descreva) a posição relativa desses vetores: $\vec{a}$ aponta para o primeiro octante; $\vec{b}$ está no plano $Oxz$. O ângulo obtido faz sentido geometricamente?
DICA DE SOBREVIVÊNCIA

$|\vec{v}| = \sqrt{v_1^2 + v_2^2 + v_3^2}$. Para o ângulo: $\cos \theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}$. Lembre-se de que $\vec{a} \cdot \vec{b} = a_1b_1 + a_2b_2 + a_3b_3$.

ACESSO NÍVEL: SUPERVISOR
  1. Módulos: $$|\vec{a}| = \sqrt{1^2 + 2^2 + 2^2} = \sqrt{1 + 4 + 4} = \sqrt{9} = 3$$ $$|\vec{b}| = \sqrt{3^2 + 0^2 + 4^2} = \sqrt{9 + 0 + 16} = \sqrt{25} = 5$$
  2. Produto interno: $$\vec{a} \cdot \vec{b} = (1)(3) + (2)(0) + (2)(4) = 3 + 0 + 8 = 11$$ $$\cos \theta = \frac{11}{3 \cdot 5} = \frac{11}{15} \approx 0{,}7333$$ $$\theta = \arccos(0{,}7333) \approx 42°$$
  3. Esboço: $\vec{a} = (1, 2, 2)$ está no primeiro octante ($x > 0, y > 0, z > 0$). $\vec{b} = (3, 0, 4)$ está no plano $Oxz$ ($y = 0$). Ambos têm componentes positivas em $x$ e $z$, então apontam para a mesma "região geral" do espaço. Um ângulo agudo de cerca de $42°$ é geometricamente plausível — os vetores não são paralelos nem perpendiculares, e ambos "inclinam para cima" no eixo $z$.

Reflexão: As fórmulas são literalmente as mesmas do R². A única complexidade adicional é calcular uma raiz com três quadrados e um produto com três termos. A interpretação geométrica (ângulo agudo quando produto interno positivo, obtuso quando negativo) permanece válida.
