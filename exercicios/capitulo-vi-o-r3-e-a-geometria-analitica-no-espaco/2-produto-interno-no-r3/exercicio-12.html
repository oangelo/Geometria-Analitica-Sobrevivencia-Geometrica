Vetor Perpendicular a Dois Vetores Dados

Objetivo: Resolver um problema aberto: encontrar um vetor unitário perpendicular a dois vetores dados no R³, introduzindo geometricamente a ideia de vetor normal a um plano.

Dados $\vec{v} = (1, 2, -1)$ e $\vec{w} = (2, -1, 1)$, e sabendo que $\vec{u}$ é um vetor unitário ($|\vec{u}| = 1$) perpendicular a ambos:

  1. Escreva o sistema de equações que $\vec{u} = (x, y, z)$ deve satisfazer.
  2. Resolva o sistema para encontrar a direção de $\vec{u}$ (a menos de um sinal).
  3. Normalize para obter $\vec{u}$ com $|\vec{u}| = 1$. Há duas soluções possíveis; apresente ambas.
  4. [Interpretação geométrica] Os vetores $\vec{v}$ e $\vec{w}$ definem um plano. O que representa geometricamente o vetor $\vec{u}$ encontrado? Por que há duas soluções?
DICA DE SOBREVIVÊNCIA

As condições são: $\vec{u} \cdot \vec{v} = 0$ e $\vec{u} \cdot \vec{w} = 0$. Isso dá duas equações lineares em três incógnitas. Resolva o sistema por escalonamento. A normalização usa $|\vec{u}| = \sqrt{x^2 + y^2 + z^2} = 1$.

ACESSO NÍVEL: SUPERVISOR
  1. Sistema de equações: $$\vec{u} \cdot \vec{v} = x + 2y - z = 0 \quad \text{(Eq. 1)}$$ $$\vec{u} \cdot \vec{w} = 2x - y + z = 0 \quad \text{(Eq. 2)}$$ $$x^2 + y^2 + z^2 = 1 \quad \text{(Eq. 3: normalização)}$$
  2. Resolvendo Eq. 1 + Eq. 2: $$3x + y = 0 \Rightarrow y = -3x$$

    Substituindo em Eq. 1: $$x + 2(-3x) - z = 0 \Rightarrow x - 6x - z = 0 \Rightarrow z = -5x$$

    Portanto, $\vec{u} = (x, -3x, -5x) = x(1, -3, -5)$. A direção é $(1, -3, -5)$ (ou qualquer múltiplo não-nulo).

  3. Normalização: $$|(1, -3, -5)| = \sqrt{1 + 9 + 25} = \sqrt{35}$$

    As duas soluções unitárias (sinais opostos):

    $$\vec{u}_1 = \frac{1}{\sqrt{35}}(1, -3, -5) = \left(\frac{1}{\sqrt{35}}, -\frac{3}{\sqrt{35}}, -\frac{5}{\sqrt{35}}\right)$$ $$\vec{u}_2 = -\vec{u}_1 = \frac{1}{\sqrt{35}}(-1, 3, 5)$$
  4. Interpretação geométrica: Os vetores $\vec{v}$ e $\vec{w}$ (que não são paralelos, pois $\vec{v} \neq \lambda \vec{w}$) definem um plano passando pela origem. O vetor $\vec{u}$ é normal (perpendicular) a este plano — ele aponta na direção "para cima" ou "para baixo" do plano. As duas soluções $\vec{u}_1$ e $\vec{u}_2$ correspondem aos dois sentidos possíveis da normal: uma de cada lado do plano. No Capítulo VII, esta operação será sistematizada como o produto vetorial: $\vec{u} \parallel \vec{v} \times \vec{w}$.

Reflexão: Este exercício antecipa o produto vetorial (Cap VII). A ideia central é: no R³, dados dois vetores não-paralelos, existe uma única direção (a menos de sinal) perpendicular a ambos. Esta direção é a "normal ao plano" definido pelos dois vetores. No R², dois vetores não-paralelos já definem todo o espaço; não há direção "fora do plano". A existência de uma terceira direção perpendicular é uma das propriedades definidoras do espaço tridimensional.
