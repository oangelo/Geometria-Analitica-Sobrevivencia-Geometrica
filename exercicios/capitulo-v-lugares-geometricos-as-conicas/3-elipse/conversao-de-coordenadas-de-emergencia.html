CONVERSÃO DE COORDENADAS DE EMERGÊNCIA

O objetivo deste exercício é converter uma equação geral da elipse para a forma canônica, e depois extrair seus elementos. Um sistema de navegação legado transmite a fronteira de uma zona segura na forma $9x^2 + 4y^2 = 36$.

(a) Divida ambos os lados da equação por 36 para obter a forma canônica $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$. Identifique $a^2$ e $b^2$.

(b) Determine se a elipse é horizontal ou vertical. Faça $y = 0$ e $x = 0$ para encontrar os vértices nos dois eixos.

(c) Calcule $c$ pela relação $c^2 = a^2 - b^2$ e encontre as coordenadas dos focos.

(d) Calcule a excentricidade $e = \frac{c}{a}$.

(e) No papel, esboce a elipse com todos os elementos.

DICA DE SOBREVIVÊNCIA

Dividir $9x^2 + 4y^2 = 36$ por 36 dá $\frac{9x^2}{36} + \frac{4y^2}{36} = 1$, que simplifica para $\frac{x^2}{4} + \frac{y^2}{9} = 1$. O maior denominador é 9, que está sob $y^2$ — portanto a elipse é vertical.

ACESSO NÍVEL: SUPERVISOR
  1. Equação geral: $9x^2 + 4y^2 = 36$.
  2. Divida por 36: $\frac{9x^2}{36} + \frac{4y^2}{36} = 1$.
  3. Simplifique: $\frac{x^2}{4} + \frac{y^2}{9} = 1$.
  4. Identifique: $a^2 = 9$ (maior denominador, sob $y^2$), $b^2 = 4$ (sob $x^2$).
  5. Logo $a = 3$, $b = 2$. A elipse é vertical.
  6. Faça $y = 0$: $\frac{x^2}{4} = 1 \Rightarrow x = \pm 2$. Vértices: $(\pm 2, 0)$.
  7. Faça $x = 0$: $\frac{y^2}{9} = 1 \Rightarrow y = \pm 3$. Vértices: $(0, \pm 3)$.
  8. Calcule $c$: $c^2 = a^2 - b^2 = 9 - 4 = 5$, logo $c = \sqrt{5}$.
  9. Focos: $F_1(0, -\sqrt{5})$ e $F_2(0, \sqrt{5})$ (aproximadamente $(0, \pm 2{,}24)$).
  10. Excentricidade: $e = \frac{c}{a} = \frac{\sqrt{5}}{3} \approx 0{,}745$.
$$\frac{x^2}{4} + \frac{y^2}{9} = 1, \quad a = 3, \quad b = 2, \quad c = \sqrt{5}, \quad e = \frac{\sqrt{5}}{3}$$

Reflexão de Sobrevivência: Sistemas legados frequentemente usam a forma geral $Ax^2 + By^2 = C$. A conversão para a forma canônica é apenas uma divisão — mas é essencial para identificar os elementos geométricos. Na Zona Devastada, saber ler diferentes formatos de dados pode ser a diferença entre encontrar ou não a zona segura.
