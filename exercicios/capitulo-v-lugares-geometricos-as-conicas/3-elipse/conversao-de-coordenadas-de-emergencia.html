Conversão de Coordenadas de Emergência

Durante uma tempestade radioativa, você recebeu as coordenadas de uma zona segura através da equação geral $4x^2 + 9y^2 = 36$. Converta para a forma canônica e identifique os elementos essenciais desta zona de proteção.

DICA DE SOBREVIVÊNCIA

Divida toda a equação por 36 para obter a forma canônica. Depois compare com o padrão para identificar a² e b².

ACESSO NÍVEL: SUPERVISOR
  1. Dividir a equação por 36: $\frac{4x^2}{36} + \frac{9y^2}{36} = 1$
  2. Simplificar: $\frac{x^2}{9} + \frac{y^2}{4} = 1$
  3. Identificar: $a^2 = 9$ (maior denominador), logo $a = 3$
  4. Identificar: $b^2 = 4$, logo $b = 2$
  5. Calcular: $c^2 = a^2 - b^2 = 9 - 4 = 5$, logo $c = \sqrt{5}$
$$\frac{x^2}{9} + \frac{y^2}{4} = 1$$ $$a = 3, \quad b = 2, \quad c = \sqrt{5}$$

Reflexão de Sobrevivência: A conversão rápida de equações é vital em emergências. Esta zona tem formato elíptico horizontal com focos em $(\pm\sqrt{5}, 0)$, oferecendo proteção máxima na direção horizontal.
