Hipérbole a partir de Focos e Vértice

O objetivo deste exercício é encontrar a equação de uma hipérbole quando conhecemos seus focos e um vértice. Dois detectores de radiação estão em $F_1(-4,0)$ e $F_2(4,0)$. Um ponto de medição confirmou que $(3,0)$ pertence à hipérbole — este é um vértice.

(a) A partir dos focos, identifique $c$. A partir do vértice $(3,0)$, identifique $a$.

(b) Use a relação $c^2 = a^2 + b^2$ para encontrar $b^2$. Não use fórmulas decoradas — deduza $b$ a partir da relação fundamental.

(c) Escreva a equação canônica da hipérbole.

(d) Verifique que o ponto $(3,0)$ satisfaz a equação. Depois verifique que $d((3,0), F_1) - d((3,0), F_2) = 2a$.

(e) Calcule as equações das assíntotas e faça um esboço no papel com todos os elementos.

DICA DE SOBREVIVÊNCIA

O vértice de uma hipérbole horizontal está em $(a, 0)$. Se o vértice dado é $(3, 0)$, então $a = 3$. Os focos são $(\pm c, 0) = (\pm 4, 0)$, então $c = 4$. Da relação $c^2 = a^2 + b^2$, isole $b^2$.

ACESSO NÍVEL: SUPERVISOR
  1. Focos em $(\pm 4, 0)$, logo $c = 4$ e $c^2 = 16$.
  2. Vértice em $(3, 0)$, logo $a = 3$ e $a^2 = 9$.
  3. Pela relação $c^2 = a^2 + b^2$: $16 = 9 + b^2$, logo $b^2 = 7$.
  4. Equação canônica: $\frac{x^2}{9} - \frac{y^2}{7} = 1$.
  5. Verificação do vértice: $\frac{3^2}{9} - \frac{0^2}{7} = \frac{9}{9} = 1$ ✓
  6. Verificação da definição: $d((3,0), F_1) = |3 - (-4)| = 7$, $d((3,0), F_2) = |3 - 4| = 1$, diferença $= 7 - 1 = 6 = 2a$ ✓
  7. Assíntotas: $y = \pm \frac{b}{a}x = \pm \frac{\sqrt{7}}{3}x$.
$$\text{Equação: } \frac{x^2}{9} - \frac{y^2}{7} = 1 \quad \text{com } a = 3,\; b = \sqrt{7},\; c = 4$$
$$\text{Assíntotas: } y = \pm \frac{\sqrt{7}}{3}x$$

Reflexão de Sobrevivência: Quando $b^2$ não é um quadrado perfeito, deixamos $b = \sqrt{7}$. Não aproxime! Nas assíntotas, $\frac{\sqrt{7}}{3} \approx 0{,}88$ é apenas uma referência para o esboço — a forma exata $\frac{\sqrt{7}}{3}$ é mais precisa.
