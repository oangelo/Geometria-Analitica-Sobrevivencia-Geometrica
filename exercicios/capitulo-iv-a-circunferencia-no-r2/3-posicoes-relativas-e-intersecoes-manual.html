> MANUAL DE SOBREVIVÊNCIA MATEMÁTICA: POSIÇÕES RELATIVAS ENTRE RETA E CIRCUNFERÊNCIA

Na Zona Devastada, a interação entre rotas e zonas seguras é crucial para nossa sobrevivência. Matematicamente, isso é análogo ao estudo das posições relativas entre retas e circunferências - um conhecimento que permite mapear zonas de perigo, estabelecer perímetros defensivos e planejar rotas de abastecimento com precisão.

Uma reta e uma circunferência no plano podem se relacionar de três maneiras fundamentais: podem não se encontrar (reta externa), podem se tocar em exatamente um ponto (reta tangente), ou podem se cruzar em dois pontos distintos (reta secante). Identificar estas relações nos permite prever encontros com mutantes e evitar radiação perigosa.

A chave para determinar a posição relativa está na distância entre o centro da circunferência e a reta. Comparando esta distância com o raio da circunferência, podemos determinar se nossa rota é segura ou se cruza zonas de alto risco.

$\text{Seja uma circunferência } (x - x_0)^2 + (y - y_0)^2 = r^2 \text{ e uma reta } ax + by + c = 0:$ $\text{A distância do centro } (x_0, y_0) \text{ à reta é: } d = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}$ $\text{• Se } d > r \text{: reta externa (0 interseções)}$ $\text{• Se } d = r \text{: reta tangente (1 interseção)}$ $\text{• Se } d < r \text{: reta secante (2 interseções)}$

Para encontrar os pontos precisos de interseção, substituímos a equação da reta na da circunferência, resultando em uma equação quadrática. As raízes desta equação representam os pontos onde os caminhos se cruzam - informação vital para evitar emboscadas nas patrulhas diárias.
