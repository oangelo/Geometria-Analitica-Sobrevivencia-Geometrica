> MANUAL DE SOBREVIVÊNCIA MATEMÁTICA: MÓDULO DE VETORES

Sobreviventes, a capacidade de medir a intensidade de forças, distâncias e direções pode significar a diferença entre vida e morte na Zona Devastada. O módulo (ou magnitude) de um vetor é uma ferramenta essencial para quantificar essas grandezas.

O módulo de um vetor $\vec{v} = (v_x, v_y)$ representa seu "tamanho" ou "comprimento" - quanto maior o módulo, maior a intensidade da grandeza representada. Seja para calcular a força necessária para empurrar um veículo enferrujado ou a quantidade de RemoveRad necessária para neutralizar a radiação de uma área, esta medida é crucial.

Para vetores no plano $\mathbb{R}^2$, o módulo é calculado através da distância euclidiana do ponto até a origem, aplicando o teorema de Pitágoras.

$$|\vec{v}| = \|\vec{v}\| = \sqrt{v_x^2 + v_y^2}$$

As propriedades fundamentais do módulo de vetores incluem:

  • O módulo é sempre não-negativo: $|\vec{v}| \geq 0$
  • O módulo é zero se, e somente se, o vetor for nulo: $|\vec{v}| = 0 \Leftrightarrow \vec{v} = \vec{0}$
  • O módulo de um vetor multiplicado por um escalar é igual ao valor absoluto do escalar multiplicado pelo módulo do vetor: $|k\vec{v}| = |k||\vec{v}|$
