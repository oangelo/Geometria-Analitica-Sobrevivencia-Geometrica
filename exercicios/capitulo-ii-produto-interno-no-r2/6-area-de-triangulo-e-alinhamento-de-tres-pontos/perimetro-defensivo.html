PERÍMETRO DEFENSIVO

Nos exercícios anteriores, usamos o determinante para calcular áreas e testar alinhamento, e o produto escalar para verificar ortogonalidade. Agora o SUPERVISOR precisa que você planeje o perímetro defensivo de um novo assentamento. O terreno disponível é delimitado por quatro postos: $A(0, 0)$, $B(6, 0)$, $C(8, 5)$ e $D(2, 5)$. O quadrilátero pode ser dividido em dois triângulos: $ABC$ e $ACD$. Calcule a área de cada triângulo usando o determinante 2×2 e some para obter a área total. Depois, verifique se os lados opostos são paralelos ($\overrightarrow{AB}$ é paralelo a $\overrightarrow{DC}$? $\overrightarrow{AD}$ é paralelo a $\overrightarrow{BC}$?). O que isso revela sobre a forma do terreno?

DICA DE SOBREVIVÊNCIA

Para a área de $ABC$: use $\overrightarrow{AB}$ e $\overrightarrow{AC}$. Para $ACD$: use $\overrightarrow{AC}$ e $\overrightarrow{AD}$. Para testar paralelismo, verifique se os vetores são proporcionais (tópico 4).

ACESSO NÍVEL: SUPERVISOR
  1. Triângulo $ABC$: $\overrightarrow{AB} = (6, 0)$ e $\overrightarrow{AC} = (8, 5)$. Área $= \frac{1}{2}|6 \times 5 - 0 \times 8| = \frac{1}{2} \times 30 = 15$.
  2. Triângulo $ACD$: $\overrightarrow{AC} = (8, 5)$ e $\overrightarrow{AD} = (2, 5)$. Área $= \frac{1}{2}|8 \times 5 - 5 \times 2| = \frac{1}{2}|40 - 10| = \frac{1}{2} \times 30 = 15$.
  3. Área total: $15 + 15 = 30$ unidades quadradas.
  4. Teste de paralelismo: $\overrightarrow{AB} = (6, 0)$ e $\overrightarrow{DC} = (8-2, 5-5) = (6, 0)$. São iguais! Paralelos ✓
  5. $\overrightarrow{AD} = (2, 5)$ e $\overrightarrow{BC} = (8-6, 5-0) = (2, 5)$. São iguais! Paralelos ✓
  6. Lados opostos paralelos → o terreno é um paralelogramo. A área também poderia ser calculada diretamente: $|\overrightarrow{AB}| \times h$ (onde $h$ é a distância vertical entre lados paralelos = $5$). Área $= 6 \times 5 = 30$ ✓.
$$\text{Área}_{ABC} = 15, \quad \text{Área}_{ACD} = 15, \quad \text{Área total} = 30$$ $$\overrightarrow{AB} = \overrightarrow{DC} = (6, 0) \implies \text{paralelogramo}$$

Reflexão de Sobrevivência: Este exercício integra tudo: determinante para área, vetores para paralelismo, e geometria para interpretação. O terreno é um paralelogramo — isso facilita o planejamento: lados opostos iguais, defesa simétrica. Na Zona Devastada, reconhecer a forma do terreno pelo cálculo vetorial permite planejar defesas antes mesmo de visitar o local.
