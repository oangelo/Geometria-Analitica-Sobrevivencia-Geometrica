INTERPRETAÇÃO DA EQUAÇÃO REDUZIDA

O sistema de navegação do Refúgio-TEC armazena rotas na forma reduzida $y = mx + n$. Uma rota de suprimentos está registrada como $y = -\frac{1}{2}x + 4$. No papel, esboce a reta marcando o intercepto no eixo $y$ e usando a inclinação para encontrar um segundo ponto. Depois, interprete geometricamente: qual é a inclinação do terreno e onde a rota cruza o eixo vertical? A rota sobe ou desce?

DICA DE SOBREVIVÊNCIA

Na forma $y = mx + n$, o número que multiplica $x$ é o coeficiente angular $m$ e o termo isolado é o coeficiente linear $n$. O intercepto no eixo $y$ é o ponto $(0, n)$. A partir dele, use $m = \frac{\Delta y}{\Delta x}$ para encontrar outro ponto.

ACESSO NÍVEL: SUPERVISOR
  1. Compare $y = -\frac{1}{2}x + 4$ com $y = mx + n$.
  2. Identifique: $m = -\frac{1}{2}$ e $n = 4$.
  3. O intercepto no eixo $y$ é o ponto $(0, 4)$.
  4. A inclinação $m = -\frac{1}{2}$ significa que para cada 2 unidades para a direita, a reta desce 1 unidade.
  5. A partir de $(0, 4)$, andando $\Delta x = 2$ e $\Delta y = -1$, chegamos a $(2, 3)$.
  6. Como $m < 0$, a rota desce da esquerda para a direita.
$$m = -\frac{1}{2}, \quad n = 4, \quad \text{intercepto: } (0, 4)$$

Reflexão de Sobrevivência: A forma reduzida $y = mx + n$ é a "ficha técnica" de uma rota: $m$ diz se é subida ou descida e quão íngreme, enquanto $n$ diz onde a rota começa no eixo vertical. Na Zona Devastada, ler esses dois números rapidamente pode ser a diferença entre escolher uma rota viável e uma que deixa o veículo atolado.
