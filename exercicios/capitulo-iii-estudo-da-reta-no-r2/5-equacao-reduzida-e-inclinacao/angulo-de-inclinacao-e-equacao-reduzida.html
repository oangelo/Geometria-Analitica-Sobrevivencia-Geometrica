ÂNGULO DE INCLINAÇÃO E EQUAÇÃO REDUZIDA

Uma rampa de acesso ao abrigo subterrâneo precisa ser construída com ângulo de inclinação de $45°$ em relação ao solo, partindo do ponto $P(0, 3)$ no eixo $y$. No papel, esboce a situação: marque o ponto $P$ e trace uma reta que forme $45°$ com a horizontal. Use a relação $m = \tan\theta$ para encontrar o coeficiente angular e depois escreva a equação reduzida da reta que representa essa rampa.

DICA DE SOBREVIVÊNCIA

O coeficiente angular é a tangente do ângulo de inclinação. Para $\theta = 45°$, use o valor de $\tan 45°$ dos ângulos notáveis do Capítulo 0. Depois, substitua $m$ e o ponto $P$ em $y = mx + n$ para encontrar $n$.

ACESSO NÍVEL: SUPERVISOR
  1. O ângulo de inclinação é $\theta = 45°$.
  2. Calcule o coeficiente angular: $m = \tan 45° = 1$.
  3. A equação reduzida tem a forma $y = 1 \cdot x + n = x + n$.
  4. Substitua o ponto $P(0, 3)$: $3 = 0 + n$, logo $n = 3$.
  5. A equação reduzida da rampa é $y = x + 3$.
$$m = \tan 45° = 1$$ $$y = x + n \xrightarrow{P(0,3)} 3 = 0 + n \Rightarrow n = 3$$ $$y = x + 3$$

Reflexão de Sobrevivência: Uma rampa com inclinação de $45°$ sobe exatamente o mesmo que avança horizontalmente — para cada metro para frente, sobe um metro. Na construção de abrigos, ângulos muito íngremes dificultam o transporte de suprimentos; ângulos muito suaves exigem rampas longas demais, expondo a entrada. O $45°$ é um compromisso clássico entre segurança e eficiência.
