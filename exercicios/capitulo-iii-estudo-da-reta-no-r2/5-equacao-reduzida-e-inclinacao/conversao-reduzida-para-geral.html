CONVERSÃO REDUZIDA PARA GERAL

Uma rota de fuga foi calculada na forma reduzida $y = -\frac{3}{4}x + 5$ pelo sistema de planejamento tático. Para alimentar o sistema de defesa automática — que só aceita a forma geral $ax + by + c = 0$ — você precisa converter essa equação. Reorganize todos os termos para um só lado, elimine frações se necessário, e apresente a equação geral com coeficientes inteiros.

DICA DE SOBREVIVÊNCIA

Mova todos os termos para o lado esquerdo da igualdade. Se houver frações, multiplique toda a equação pelo denominador comum para obter coeficientes inteiros. A forma geral exige que o lado direito seja zero.

ACESSO NÍVEL: SUPERVISOR
  1. Parta da forma reduzida: $y = -\frac{3}{4}x + 5$.
  2. Mova todos os termos para a esquerda: $\frac{3}{4}x + y - 5 = 0$.
  3. Elimine a fração multiplicando por 4: $3x + 4y - 20 = 0$.
  4. Verifique: $a = 3$, $b = 4$, $c = -20$, todos inteiros, lado direito igual a zero.
$$y = -\frac{3}{4}x + 5$$ $$\frac{3}{4}x + y - 5 = 0$$ $$\times 4: \quad 3x + 4y - 20 = 0$$

Reflexão de Sobrevivência: Diferentes sistemas exigem diferentes formatos. O planejador tático pensa em $y = mx + n$ porque $m$ e $n$ têm significado imediato, mas o sistema de defesa trabalha com a forma geral porque ela unifica o tratamento de todas as retas — inclusive as verticais. Saber traduzir entre os dois é como falar duas línguas na Zona Devastada.
