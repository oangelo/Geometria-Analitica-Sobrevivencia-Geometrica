INTERPRETAÇÃO GEOMÉTRICA DO SISTEMA

Um técnico do Refúgio-29 analisa três sistemas 2×2 e precisa interpretar geometricamente o que cada um significa sobre as retas envolvidas. Para cada sistema abaixo, classifique-o como possível e determinado, possível e indeterminado, ou impossível. Depois, traduza essa classificação para a posição relativa das retas (concorrentes, coincidentes ou paralelas) e descreva o que isso significa geometricamente — há um único ponto de encontro, infinitos pontos em comum, ou nenhum?

(a) $\begin{cases} x + y = 5 \\ 2x + 2y = 10 \end{cases}$

(b) $\begin{cases} x + y = 3 \\ x + y = 7 \end{cases}$

(c) $\begin{cases} 2x + y = 7 \\ x - y = -1 \end{cases}$

DICA DE SOBREVIVÊNCIA

Observe os coeficientes: no sistema (a), a segunda equação é o dobro da primeira. No (b), os lados esquerdos são idênticos mas os direitos diferem. No (c), os coeficientes de $x$ e $y$ são independentes — tente resolver por eliminação.

ACESSO NÍVEL: SUPERVISOR
  1. Sistema (a): A segunda equação é exatamente o dobro da primeira. Dividindo a segunda por 2 obtemos $x + y = 5$, que é idêntica à primeira. O sistema é possível e indeterminado — infinitas soluções. Geometricamente, as retas são coincidentes: são a mesma reta, então "se cruzam" em todos os seus pontos.
  2. Sistema (b): Subtraindo as equações: $(x + y) - (x + y) = 3 - 7$, ou seja, $0 = -4$. Contradição. O sistema é impossível — nenhuma solução. Geometricamente, as retas são paralelas distintas: mantêm a mesma inclinação mas nunca se tocam.
  3. Sistema (c): Somando as duas equações: $3x = 6$, logo $x = 2$. Substituindo na segunda: $2 - y = -1$, logo $y = 3$. O sistema é possível e determinado — solução única $(2, 3)$. Geometricamente, as retas são concorrentes: se cruzam em exatamente um ponto.
$$\text{(a) Coincidentes: } \frac{1}{2} = \frac{1}{2} = \frac{5}{10} \quad \text{(b) Paralelas: } \frac{1}{1} = \frac{1}{1} \neq \frac{3}{7} \quad \text{(c) Concorrentes: } \frac{2}{1} \neq \frac{1}{-1}$$

Reflexão de Sobrevivência: A classificação do sistema 2×2 é o "scanner" de rotas: determinado significa "há um ponto de encontro exato", indeterminado significa "os dois relatórios descrevem a mesma trilha" e impossível significa "as rotas nunca se cruzam — não perca tempo procurando". Na Zona Devastada, interpretar corretamente o sistema evita expedições inúteis.
