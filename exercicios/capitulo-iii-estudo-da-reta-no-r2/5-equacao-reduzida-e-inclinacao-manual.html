> MANUAL DE SOBREVIVÊNCIA MATEMÁTICA: EQUAÇÃO REDUZIDA E INCLINAÇÃO

Na Zona Devastada, terrenos íngremes, rampas de acesso e linhas de visada são questões de vida ou morte. Uma rampa muito inclinada derruba um veículo de carga; uma linha de tiro mal calculada desperdiça munição. Precisamos de uma forma de medir o quanto uma reta "sobe" ou "desce" — e essa medida é o coeficiente angular.

Dados dois pontos $A(x_1, y_1)$ e $B(x_2, y_2)$ de uma reta, o coeficiente angular $m$ mede a taxa de variação vertical por unidade horizontal:

$$m = \frac{\Delta y}{\Delta x} = \frac{y_2 - y_1}{x_2 - x_1}$$

O sinal de $m$ indica a direção: positivo significa subida (esquerda para direita), negativo significa descida, e zero significa terreno plano. Quando $\Delta x = 0$, a reta é vertical e o coeficiente angular não existe — a inclinação é de $90°$.

O coeficiente angular também se relaciona com o ângulo de inclinação $\theta$ que a reta forma com o eixo $x$ positivo:

$$m = \tan\theta$$

Com isso, podemos traduzir entre a linguagem algébrica ($m$) e a linguagem angular ($\theta$). Ângulos notáveis do Capítulo 0 reaparecem aqui: $\tan 30° = \frac{\sqrt{3}}{3}$, $\tan 45° = 1$, $\tan 60° = \sqrt{3}$.

A equação reduzida da reta isola $y$ e revela dois números de uma só vez:

$$y = mx + n$$

Aqui, $m$ é o coeficiente angular (inclinação) e $n$ é o coeficiente linear (intercepto no eixo $y$). Essa forma é especialmente útil para leitura rápida: olhando para $y = 2x + 3$, sabemos imediatamente que a reta sobe 2 unidades para cada 1 unidade horizontal e cruza o eixo $y$ no ponto $(0, 3)$.

Para converter da equação geral $ax + by + c = 0$ para a reduzida, isolamos $y$:

$$by = -ax - c \quad \Rightarrow \quad y = -\frac{a}{b}x - \frac{c}{b}$$

Portanto, $m = -\frac{a}{b}$ e $n = -\frac{c}{b}$. No caminho inverso, partimos de $y = mx + n$ e rearranjamos para $mx - y + n = 0$, que é da forma geral com $a = m$, $b = -1$, $c = n$.

O coeficiente angular governa as posições relativas entre retas. Duas retas são paralelas quando têm a mesma inclinação ($m_1 = m_2$) e perpendiculares quando o produto de suas inclinações é $-1$ ($m_1 \cdot m_2 = -1$). Com isso, podemos verificar alinhamento de rotas ou construir defesas em ângulo reto sem sair da equação reduzida.

Técnicas de sobrevivência:

1. Dados dois pontos, calcule $m$ primeiro; depois substitua um ponto em $y = mx + n$ para encontrar $n$.

2. Dado $\theta$, use $m = \tan\theta$ para obter a inclinação sem calcular pontos.

3. Na conversão geral $\to$ reduzida, verifique se $b \neq 0$; caso contrário, a reta é vertical.

4. Para verificar paralelismo ou perpendicularidade, compare apenas os coeficientes angulares.
