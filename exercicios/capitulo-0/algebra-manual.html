Fase 1 — Isolamento de incógnitas (equações lineares)

Toda equação algébrica nasce de uma situação prática: conhecemos algumas grandezas e precisamos descobrir outras. O princípio fundamental é que a mesma operação aplicada aos dois lados mantém a igualdade válida. Começamos com equações simples — apenas uma operação envolvendo $x$ — e avançamos para equações com múltiplas operações.

$$x + a = b \implies x = b - a$$ $$ax = b \implies x = \frac{b}{a}$$ $$ax + b = c \implies x = \frac{c - b}{a}$$ $$\frac{x}{a} + b = c \implies x = a(c - b)$$

Regra de ouro: quando há múltiplas operações envolvendo $x$, desfaça na ordem inversa à que foi aplicada.

Fase 2 — Propriedades algébricas

Antes de resolver equações mais complexas, precisamos dominar as ferramentas que facilitarão o trabalho. Os produtos notáveis e as propriedades de potências são padrões que se repetem com frequência — reconhecê-los economiza tempo e evita erros.

$$(a + b)^2 = a^2 + 2ab + b^2$$ $$(a - b)^2 = a^2 - 2ab + b^2$$ $$(a + b)(a - b) = a^2 - b^2$$ $$a^m \times a^n = a^{m+n}$$ $$a^m \div a^n = a^{m-n}$$

Erro comum: $(a+b)^2 \neq a^2 + b^2$. O termo $2ab$ não pode ser ignorado. Geometricamente, ele representa dois retângulos de área $ab$ que ficam "de fora" quando somamos os quadrados de $a$ e $b$.

Fase 3 — Combinar ferramentas com incógnitas

Agora combinamos o que aprendemos: usamos produtos notáveis para expandir expressões e depois resolvemos a equação resultante com técnicas de isolamento. Reconhecer padrões — como a diferença de quadrados — pode transformar uma equação aparentemente complexa em algo simples.

Fase 4 — Aplicação

Nos exercícios finais, a equação não é dada — cabe a você traduzir a situação real em equação e resolvê-la. Este é o objetivo final: não apenas resolver equações, mas criá-las.

Domine estas ferramentas e você sobreviverá onde outros falharão. Erros não são uma opção quando cada cálculo pode significar vida ou morte.
