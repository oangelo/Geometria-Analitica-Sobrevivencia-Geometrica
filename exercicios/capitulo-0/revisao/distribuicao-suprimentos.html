MISSÃO 4: Distribuição de Suprimentos

Três abrigos compartilham recursos no total de $210$ unidades de comida e $225$ unidades de medicamentos. O primeiro abrigo recebe $2x + 3y$ unidades de comida e $3x - y$ de medicamentos; o segundo recebe $x - 2y$ de comida e $4x + 2y$ de medicamentos; o terceiro recebe o restante: $4x + 4y$ de comida e $2x + 4y$ de medicamentos. Some as expressões de cada recurso entre os três abrigos, monte um sistema de equações e resolva para descobrir os valores de $x$ e $y$ que tornam a distribuição consistente com o total disponível.

ACESSO NÍVEL: SUPERVISOR
  1. Somamos as expressões de comida e medicamentos para montar o sistema:
    $$\text{Comida: } (2x + 3y) + (x - 2y) + (4x + 4y) = 7x + 5y = 210$$
    $$\text{Medicamentos: } (3x - y) + (4x + 2y) + (2x + 4y) = 9x + 5y = 225$$
  2. Subtraímos a primeira equação da segunda:
    $$(9x + 5y) - (7x + 5y) = 225 - 210 \implies 2x = 15 \implies x = 7{,}5$$
  3. Substituímos em $7x + 5y = 210$:
    $$7(7{,}5) + 5y = 210 \implies 52{,}5 + 5y = 210 \implies y = 31{,}5$$
  4. Total por abrigo (comida + medicamentos): Abrigo 1 = $100{,}5$; Abrigo 2 = $37{,}5$; Abrigo 3 = $297$.

Reflexão de Sobrevivência: A distribuição desproporcional de recursos pode gerar tensões entre comunidades na Zona Devastada. Para manter alianças estáveis, avalie se o Abrigo 3 realmente tem mais sobreviventes ou está em situação mais crítica antes de aceitar a disparidade.
