MISSÃO 3: Sistema de Defesa Proporcional

Seu abrigo precisa avaliar a eficiência do sistema de defesa. Contra mutantes pequenos, a eficiência é de $\frac{1}{2}$ (meio). Contra mutantes grandes, também é $\frac{1}{2}$ (meio). A área de patrulha pode ser dividida em 5 seções iguais: 3 são zona de mutantes pequenos e 2 são zona de mutantes grandes. Como cada seção tem o mesmo tamanho e o mesmo número de mutantes, a eficiência geral é uma média ponderada — multiplique cada eficiência pela fração da área correspondente e some os resultados para descobrir se o abrigo está seguro.

ACESSO NÍVEL: SUPERVISOR
  1. Identificamos as proporções da área:
    $$\text{Mutantes pequenos: } \frac{3}{5} \text{ do total} \qquad \text{Mutantes grandes: } \frac{2}{5} \text{ do total}$$
  2. Calculamos a eficiência geral com média ponderada:
    $$E = \frac{1}{2} \times \frac{3}{5} + \frac{1}{2} \times \frac{2}{5} = \frac{3}{10} + \frac{2}{10} = \frac{5}{10} = \frac{1}{2} = 50\%$$

Reflexão de Sobrevivência: Uma eficiência de $50\%$ é preocupante. Na Zona Devastada, sistemas de defesa com menos de $75\%$ são considerados vulneráveis. Recomenda-se melhorar o sistema ou limitar explorações a áreas com menor incidência de mutantes até que o sistema seja aprimorado.
