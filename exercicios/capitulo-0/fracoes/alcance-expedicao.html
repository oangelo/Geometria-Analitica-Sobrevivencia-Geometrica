ALCANCE DA EXPEDIÇÃO

Para preparar a solução de RemoveRad corretamente, o protocolo do SUPERVISOR exige uma diluição precisa: 2 mL de RemoveRad concentrado para cada 500 mL de água destilada. Você precisa preparar 2 litros (2000 mL) dessa solução. O raciocínio é o mesmo do exercício anterior sobre proporções — montar a igualdade de razões e resolver com produto cruzado — mas agora o contexto é uma diluição real, onde um erro pode significar intoxicação ou ineficácia. Quantos mL de RemoveRad concentrado você precisará?

DICA DE SOBREVIVÊNCIA

Monte a proporção: $\frac{2}{500} = \frac{x}{2000}$. Resolva com produto cruzado.

ACESSO NÍVEL: SUPERVISOR
  1. Montamos a proporção, onde $x$ é a quantidade de RemoveRad concentrado necessária para 2000 mL de solução:
    $$\frac{2}{500} = \frac{x}{2000}$$
  2. Aplicamos o produto cruzado:
    $$2 \times 2000 = 500 \times x$$ $$4000 = 500x$$
  3. Isolamos $x$:
    $$x = \frac{4000}{500} = 8$$

Você precisará de 8 mL de RemoveRad concentrado para preparar 2 litros de solução.

Reflexão de Sobrevivência: Uma diluição errada de RemoveRad pode ser tão perigosa quanto a radiação que ele deveria tratar. Concentração alta causa intoxicação; concentração baixa é ineficaz. Na Zona Devastada, não há espaço para "aproximadamente certo" — a proporção exata é a diferença entre cura e veneno.
