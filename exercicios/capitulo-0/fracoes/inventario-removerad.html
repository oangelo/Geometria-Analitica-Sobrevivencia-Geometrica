INVENTÁRIO DE REMOVERAD

No exercício anterior, os reservatórios tinham o mesmo denominador — partes do mesmo tamanho. Mas a expedição retornou com dois frascos de RemoveRad: um contém $\frac{1}{3}$ de dose, o outro $\frac{1}{4}$. São "terços" e "quartos" — partes diferentes. Para somar, precisamos convertê-las em partes do mesmo tamanho. Lembre-se: multiplicar o numerador e o denominador pelo mesmo número não muda a fração, pois equivale a multiplicar por 1. Encontre um denominador comum, converta e some para descobrir quantas doses completas você tem ao todo.

DICA DE SOBREVIVÊNCIA

Encontre o mínimo múltiplo comum (MMC) entre 3 e 4. Depois converta ambas as frações para esse denominador comum.

ACESSO NÍVEL: SUPERVISOR
  1. Os denominadores são diferentes ($3$ e $4$). Precisamos do MMC:
    $$\text{MMC}(3, 4) = 12$$
  2. Convertemos ambas as frações para denominador $12$:
    $$\frac{1}{3} = \frac{4}{12} \quad \text{e} \quad \frac{1}{4} = \frac{3}{12}$$
  3. Agora somamos as frações com denominador comum:
    $$\frac{4}{12} + \frac{3}{12} = \frac{7}{12}$$

Você tem $\frac{7}{12}$ de dose completa de RemoveRad.

Reflexão de Sobrevivência: Na prática, $\frac{7}{12}$ de dose pode ser insuficiente para uma descontaminação completa. Saber que você precisa de mais $\frac{5}{12}$ para completar uma dose plena permite planejar a próxima expedição com precisão cirúrgica.
