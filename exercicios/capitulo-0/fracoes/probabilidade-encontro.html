PROPORÇÃO DE RECURSOS

Até agora, operávamos com frações isoladas. Mas na Zona Devastada, o mais comum é comparar grandezas: "para cada 3 unidades de água, precisamos de 5 de alimento." Quando duas frações são iguais, temos uma proporção — e a ferramenta para resolvê-la é o produto cruzado. Não é uma regra decorada: é consequência direta de que as duas frações representam a mesma razão.

O protocolo de racionamento do abrigo estabelece que a proporção entre água e alimento deve ser constante. Se para cada 3 litros de água são necessários 5 quilos de alimento, e o estoque atual tem 6 litros de água, quantos quilos de alimento são necessários para manter a proporção?

DICA DE SOBREVIVÊNCIA

Monte a proporção $\frac{3}{5} = \frac{6}{x}$ e use o produto cruzado para resolver.

ACESSO NÍVEL: SUPERVISOR
  1. Montamos a proporção, onde $x$ é a quantidade de alimento necessária:
    $$\frac{3}{5} = \frac{6}{x}$$
  2. Aplicamos o produto cruzado (multiplicação cruzada), que é consequência de ambas as frações serem iguais:
    $$3 \times x = 5 \times 6$$ $$3x = 30$$
  3. Isolamos $x$:
    $$x = \frac{30}{3} = 10$$

São necessários 10 quilos de alimento para manter a proporção com 6 litros de água.

Reflexão de Sobrevivência: Manter proporções corretas entre recursos é vital. Água sem alimento causa desnutrição; alimento sem água causa desidratação. Na Zona Devastada, quem entende proporções mantém o grupo vivo. Quem não entende, improvisa — e improvisação sem cálculo é roleta russa.
