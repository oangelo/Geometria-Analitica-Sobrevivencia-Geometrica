ISOLAMENTO DE RECURSOS

Em engenharia e ciência, é comum conhecermos algumas grandezas e descrevermos outras. Quando traduzimos essa situação para a linguagem matemática, caímos numa equação. Na Zona Devastada, a equação mais simples possível pode representar uma situação de vida ou morte: quantos litros de RemoveRad sobram após uma doação? O princípio fundamental é este: se fizermos a mesma operação dos dois lados da equação, a relação entre as grandezas continua válida — e podemos isolar a quantidade desconhecida. Vamos praticar com a equação mais simples possível.

Após uma expedição, o Abrigo recebeu uma doação de suprimentos. Após entregar 3 unidades de RemoveRad ao posto avançado, restaram 7 unidades no estoque. Se $x$ representa a quantidade inicial de RemoveRad no Abrigo, resolva a equação $x + 3 = 7$ para descobrir quantas unidades o Abrigo tinha antes da doação.

DICA DE SOBREVIVÊNCIA

Para isolar $x$, subtraia 3 dos dois lados da equação. A mesma operação dos dois lados mantém a igualdade válida.

ACESSO NÍVEL: SUPERVISOR
  1. A equação é $x + 3 = 7$. Para isolar $x$, precisamos eliminar o $+3$ do lado esquerdo.
  2. Subtraia 3 dos dois lados: $x + 3 - 3 = 7 - 3$
  3. Simplifique: $x = 4$
  4. Verificação: $4 + 3 = 7$ ✓
$$x + 3 = 7 \implies x = 4$$

Reflexão de Sobrevivência: O princípio de fazer a mesma operação dos dois lados é o fundamento de toda manipulação algébrica. Sem ele, não é possível resolver nenhuma equação — e sem equações, não há como calcular rações, doses de medicamentos ou distâncias seguras na Zona Devastada.
