MÚLTIPLOS COMBATENTES

No exercício anterior, a incógnita estava somada a um número — uma situação simples. Agora a incógnita está multiplicada por um número: imagine que cada membro da patrulha carrega a mesma quantidade $x$ de rações, e há 2 membros na patrulha. O princípio é o mesmo — mesma operação dos dois lados — mas a operação inversa agora é a divisão. Note como o raciocínio não mudou, apenas a ferramenta.

Uma patrulha de 2 sobreviventes carrega coletivamente 10 rações de emergência. Se cada membro carrega a mesma quantidade $x$, resolva a equação $2x = 10$ para descobrir quantas rações cada um carrega.

DICA DE SOBREVIVÊNCIA

Para isolar $x$, divida os dois lados da equação por 2. A multiplicação de $x$ por 2 é desfeita pela divisão.

ACESSO NÍVEL: SUPERVISOR
  1. A equação é $2x = 10$. Para isolar $x$, precisamos eliminar o fator 2.
  2. Divida os dois lados por 2: $\frac{2x}{2} = \frac{10}{2}$
  3. Simplifique: $x = 5$
  4. Verificação: $2 \times 5 = 10$ ✓
$$2x = 10 \implies x = 5$$

Reflexão de Sobrevivência: Saber resolver equações lineares é essencial para distribuir recursos equitativamente. Na Zona Devastada, uma distribuição injusta de rações pode causar conflitos internos — e conflitos internos destroem abrigos mais rápido que mutantes.
