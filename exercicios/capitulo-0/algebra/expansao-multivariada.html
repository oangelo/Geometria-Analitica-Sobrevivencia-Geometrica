DOSE DE EMERGÊNCIA: APLICAÇÃO FINAL

Este é o último exercício da lista — e o mais completo. Aqui você precisará combinar produto notável, propriedade distributiva, simplificação e isolamento de incógnita para resolver um problema real. O SUPERVISOR não dá dicas em situações de emergência. É hora de provar que você domina as ferramentas.

O laboratório do Abrigo sintetiza RemoveRad em lotes. A quantidade de RemoveRad produzida por lote é dada pela fórmula $Q = (x + 2)^2 - x^2$, onde $x$ é a quantidade de matéria-prima em quilogramas.

(a) Simplifique a expressão $(x + 2)^2 - x^2$ usando produtos notáveis.

(b) Se o laboratório precisa produzir exatamente 20 unidades de RemoveRad, resolva a equação simplificada para encontrar $x$.

(c) Interprete: por que a expressão simplificada é linear, mesmo sendo composta de termos quadráticos?

DICA DE SOBREVIVÊNCIA

Para (a), expanda $(x+2)^2 = x^2 + 4x + 4$ e depois subtraia $x^2$. Os termos $x^2$ se cancelam. Para (b), resolva a equação linear resultante.

ACESSO NÍVEL: SUPERVISOR
  1. Parte (a): expanda $(x+2)^2$ usando o produto notável: $x^2 + 2(x)(2) + 4 = x^2 + 4x + 4$
  2. Subtraia $x^2$: $(x+2)^2 - x^2 = x^2 + 4x + 4 - x^2 = 4x + 4$
  3. A expressão simplificada é $4x + 4$ — que é linear!
  4. Parte (b): resolva $4x + 4 = 20$. Subtraia 4: $4x = 16$. Divida por 4: $x = 4$ kg
  5. Verificação: $(4+2)^2 - 4^2 = 36 - 16 = 20$ ✓
  6. Parte (c): os termos quadráticos $x^2$ se cancelam porque ambos são positivos com mesmo coeficiente. É o mesmo princípio da diferença de quadrados — o padrão $a^2 - b^2$ com $a = (x+2)$ e $b = x$.
$$(x+2)^2 - x^2 = 4x + 4 = 20 \implies x = 4$$

Reflexão de Sobrevivência: Saber que $(x+2)^2 - x^2$ se reduz a $4x + 4$ é exatamente o tipo de conhecimento que separa o sobrevivente comum do técnico de laboratório indispensável.
