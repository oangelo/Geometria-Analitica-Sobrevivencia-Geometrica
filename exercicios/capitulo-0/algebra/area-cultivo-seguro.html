Agora juntamos duas ferramentas: o produto notável que descobrimos com números e o isolamento de $x$ que praticamos nas missões anteriores. Primeiro expandimos $(x+3)^2$ usando o padrão que verificamos. Depois, a equação resultante é resolvível com o método que já dominamos. A matemática avança combinando o que já sabemos.

Resolva a equação $(x + 3)^2 = 25$. Expanda o quadrado do binômio usando o produto notável e depois isole $x$.

DICA DE SOBREVIVÊNCIA

Use $(a+b)^2 = a^2 + 2ab + b^2$ para expandir $(x+3)^2$. Depois, resolva a equação resultante como uma equação quadrática: primeiro tire a raiz quadrada dos dois lados.

ACESSO NÍVEL: SUPERVISOR
  1. Expanda $(x+3)^2$ usando o produto notável: $(x+3)^2 = x^2 + 2(x)(3) + 3^2 = x^2 + 6x + 9$
  2. A equação fica: $x^2 + 6x + 9 = 25$
  3. Subtraia 25 dos dois lados: $x^2 + 6x - 16 = 0$
  4. Ou, mais elegante: perceba que $(x+3)^2 = 25$ implica $x + 3 = \pm 5$
  5. Se $x + 3 = 5$: $x = 2$
  6. Se $x + 3 = -5$: $x = -8$
  7. Verificação: $(2+3)^2 = 25$ ✓ e $(-8+3)^2 = (-5)^2 = 25$ ✓
$$(x+3)^2 = 25 \implies x + 3 = \pm 5 \implies x = 2 \text{ ou } x = -8$$

Reflexão de Sobrevivência: Quando o SUPERVISOR diz que a área de cultivo deve ser 25 unidades quadradas e o formato é um quadrado com lado $(x+3)$, existem dois valores possíveis para $x$.
