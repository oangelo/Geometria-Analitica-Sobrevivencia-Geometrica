POTÊNCIAS COM INCÓGNITAS

A regra de potências que consolidamos com números agora aparece com incógnitas. Simplificar $\frac{x^5}{x^2}$ usando a propriedade da divisão de potências — subtrair os expoentes — reduz uma expressão aparentemente complexa a algo simples. Depois, resolva $x^3 = 27$: para essa equação, você precisará de uma operação inversa que ainda não usamos — pense no "oposto" de elevar ao cubo.

DICA DE SOBREVIVÊNCIA

Para (a), subtraia os expoentes: $x^5 \div x^2 = x^{5-2}$. Para (b), pense: "qual número elevado ao cubo dá 27?" — isso é a raiz cúbica.

ACESSO NÍVEL: SUPERVISOR
  1. Parte (a): $\frac{x^5}{x^2} = x^{5-2} = x^3$
  2. Parte (b): $x^3 = 27$. A operação inversa de elevar ao cubo é a raiz cúbica.
  3. Aplique a raiz cúbica dos dois lados: $\sqrt[3]{x^3} = \sqrt[3]{27}$
  4. Isso dá: $x = 3$
  5. Verificação: $3^3 = 27$ ✓
$$\frac{x^5}{x^2} = x^3, \qquad x^3 = 27 \implies x = 3$$

Reflexão de Sobrevivência: O padrão é constante: para desfazer uma operação, aplicamos a inversa. Soma → subtração. Multiplicação → divisão. Potência → raiz. Se você dominar essa lógica, nenhuma equação será impossível de resolver — e nenhuma dose de RemoveRad será calculada errada.
