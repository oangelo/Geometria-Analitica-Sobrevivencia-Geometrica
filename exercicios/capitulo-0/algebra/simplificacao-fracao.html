FRAÇÃO COMO DISFARCE

O nível de radiação em um setor é medido por uma equação de calibração. Após ajustar o sensor, a leitura resultou na equação $\frac{x}{4} + 2 = 5$, onde $x$ é a leitura bruta em mSv. Note que $x$ aparece dividido por 4 — o mesmo que multiplicado por $\frac{1}{4}$ — mas o raciocínio é o mesmo: mesma operação dos dois lados. Reconhecer que $\frac{x}{4}$ significa "um quarto de $x$" e que a operação inversa é multiplicar por 4 é a chave. Resolva para descobrir a leitura real do sensor.

DICA DE SOBREVIVÊNCIA

Primeiro, subtraia 2 dos dois lados para ficar com apenas a fração. Depois, multiplique os dois lados por 4 para eliminar o denominador.

ACESSO NÍVEL: SUPERVISOR
  1. A equação é $\frac{x}{4} + 2 = 5$.
  2. Passo 1 — eliminar o $+2$: subtraia 2 dos dois lados: $\frac{x}{4} + 2 - 2 = 5 - 2$, ou seja, $\frac{x}{4} = 3$
  3. Passo 2 — eliminar a divisão por 4: multiplique os dois lados por 4: $\frac{x}{4} \times 4 = 3 \times 4$
  4. Simplifique: $x = 12$
  5. Verificação: $\frac{12}{4} + 2 = 3 + 2 = 5$ ✓
$$\frac{x}{4} + 2 = 5 \implies \frac{x}{4} = 3 \implies x = 12$$

Reflexão de Sobrevivência: Calibrar sensores de radiação com precisão é uma habilidade vital. Um erro na interpretação de frações pode levar a uma leitura incorreta — e uma leitura incorreta pode enviar uma expedição para dentro de uma zona letal achando que está segura.
