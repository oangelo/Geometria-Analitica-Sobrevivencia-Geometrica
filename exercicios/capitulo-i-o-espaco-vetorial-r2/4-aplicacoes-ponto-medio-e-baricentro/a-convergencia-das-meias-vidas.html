A CONVERGÊNCIA DAS MEIAS-VIDAS

Quatro estações de monitoramento formam um quadrilátero com vértices $A(0, 0)$, $B(8, 0)$, $C(9, 7)$ e $D(1, 7)$. Para criar uma rede de comunicação eficiente, planeja-se unir os pontos médios dos lados consecutivos, formando um novo quadrilátero. Prove que este novo quadrilátero é sempre um paralelogramo, verificando se os lados opostos são descritos pelo mesmo vetor de deslocamento (ou seu oposto).

DICA DE SOBREVIVÊNCIA

Calcule os quatro pontos médios, depois calcule os vetores que descrevem cada lado do novo quadrilátero. Dois lados são opostos se seus vetores são iguais ou opostos.

ACESSO NÍVEL: SUPERVISOR
  1. Pontos médios: $M_{AB} = (4, 0)$, $M_{BC} = (8{,}5, 3{,}5)$, $M_{CD} = (5, 7)$, $M_{DA} = (0{,}5, 3{,}5)$.
  2. Vetores dos lados do novo quadrilátero: $\vec{M_{AB}M_{BC}} = (8{,}5 - 4, 3{,}5 - 0) = (4{,}5, 3{,}5)$ $\vec{M_{BC}M_{CD}} = (5 - 8{,}5, 7 - 3{,}5) = (-3{,}5, 3{,}5)$ $\vec{M_{CD}M_{DA}} = (0{,}5 - 5, 3{,}5 - 7) = (-4{,}5, -3{,}5)$ $\vec{M_{DA}M_{AB}} = (4 - 0{,}5, 0 - 3{,}5) = (3{,}5, -3{,}5)$
  3. Verificando lados opostos: $\vec{M_{AB}M_{BC}} = (4{,}5, 3{,}5)$ e $\vec{M_{CD}M_{DA}} = (-4{,}5, -3{,}5)$ — são opostos!
  4. Verificando o outro par: $\vec{M_{BC}M_{CD}} = (-3{,}5, 3{,}5)$ e $\vec{M_{DA}M_{AB}} = (3{,}5, -3{,}5)$ — também são opostos!
  5. Como os pares de lados opostos são descritos por vetores opostos, o quadrilátero formado pelos pontos médios é sempre um paralelogramo.
$$\vec{M_{AB}M_{BC}} = -\vec{M_{CD}M_{DA}} = (4{,}5, 3{,}5) \text{ e } (-4{,}5, -3{,}5)$$ $$\vec{M_{BC}M_{CD}} = -\vec{M_{DA}M_{AB}} = (-3{,}5, 3{,}5) \text{ e } (3{,}5, -3{,}5)$$ $$\Rightarrow \text{Paralelogramo}$$

Reflexão de Sobrevivência: A descoberta de que pontos médios de quadriláteros sempre formam paralelogramos garante estabilidade estrutural previsível, mesmo em terrenos irregulares. Esta propriedade invariante é valiosa para engenheiros de sobrevivência construindo estruturas defensivas com recursos limitados — a geometria básica (operações com pares ordenados) é suficiente para prová-la, sem necessidade de fórmulas complexas.
