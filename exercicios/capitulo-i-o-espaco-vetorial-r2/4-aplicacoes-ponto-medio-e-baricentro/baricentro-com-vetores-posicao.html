BARICENTRO COM VETORES POSIÇÃO

Em física, o vetor que vai da origem $O(0, 0)$ até um ponto $P(x, y)$ é chamado de vetor posição e denotado por $\vec{OP} = (x, y)$. Ponto e vetor posição são "a mesma coisa de dois jeitos". Três patrulhas estão em $A(-2, 1)$, $B(4, 7)$ e $C(4, -5)$. Calcule o baricentro usando diretamente os vetores posição $\vec{OA}$, $\vec{OB}$ e $\vec{OC}$, aplicando a fórmula $G = \frac{\vec{OA} + \vec{OB} + \vec{OC}}{3}$.

DICA DE SOBREVIVÊNCIA

O vetor posição de $A(x, y)$ é simplesmente $(x, y)$. Some os três vetores posição componente a componente e divida por 3. O resultado é o vetor posição do baricentro.

ACESSO NÍVEL: SUPERVISOR
  1. Identificando os vetores posição: $\vec{OA} = (-2, 1)$ $\vec{OB} = (4, 7)$ $\vec{OC} = (4, -5)$
  2. Somando os vetores posição: $\vec{OA} + \vec{OB} + \vec{OC} = (-2 + 4 + 4, 1 + 7 + (-5)) = (6, 3)$.
  3. Calculando o baricentro: $G = \frac{(6, 3)}{3} = \left(\frac{6}{3}, \frac{3}{3}\right) = (2, 1)$.
  4. O vetor posição do baricentro é $\vec{OG} = (2, 1)$, o que significa que $G = (2, 1)$.
  5. Verificação pelo método tradicional: $G = \left(\frac{-2 + 4 + 4}{3}, \frac{1 + 7 + (-5)}{3}\right) = (2, 1)$. Os dois métodos dão o mesmo resultado.
$$\vec{OA} + \vec{OB} + \vec{OC} = (-2, 1) + (4, 7) + (4, -5) = (6, 3)$$ $$\vec{OG} = \frac{\vec{OA} + \vec{OB} + \vec{OC}}{3} = \frac{(6, 3)}{3} = (2, 1)$$ $$\Rightarrow G = (2, 1)$$

Reflexão de Sobrevivência: Pensar em termos de vetores posição unifica a linguagem: um ponto não é apenas uma "localização", mas um "deslocamento" a partir da origem. Na Zona Devastada, esta perspectiva facilita cálculos — somar vetores é mais intuitivo que manipular coordenadas isoladas. O baricentro como "média dos vetores posição" é uma formulação elegante e poderosa.
