REDE DE ABASTECIMENTO

Três comunidades estão localizadas nos pontos $A(2, 1)$, $B(10, 1)$ e $C(6, 8)$. Para economizar recursos, uma central de distribuição será construída no baricentro deste triângulo. O baricentro tem uma propriedade notável: ele minimiza a soma das coordenadas deslocadas de cada vértice. Verifique isso calculando o baricentro $G$ e comparando as coordenadas de $G$ com a média aritmética das coordenadas de cada vértice.

(a) Calcule o baricentro $G$.

(b) Calcule as médias individuais: $\bar{x} = \frac{x_A + x_B + x_C}{3}$ e $\bar{y} = \frac{y_A + y_B + y_C}{3}$. O que você observa?

DICA DE SOBREVIVÊNCIA

O baricentro é definido como a média aritmética das coordenadas dos vértices. Verifique se $G_x = \bar{x}$ e $G_y = \bar{y}$.

ACESSO NÍVEL: SUPERVISOR
  1. Baricentro: $G = \left(\frac{2 + 10 + 6}{3}, \frac{1 + 1 + 8}{3}\right) = \left(\frac{18}{3}, \frac{10}{3}\right) = (6, \frac{10}{3}) \approx (6, 3{,}33)$.
  2. Média das coordenadas $x$: $\bar{x} = \frac{2 + 10 + 6}{3} = 6$.
  3. Média das coordenadas $y$: $\bar{y} = \frac{1 + 1 + 8}{3} = \frac{10}{3} \approx 3{,}33$.
  4. Observação: $G = (\bar{x}, \bar{y})$. O baricentro é exatamente a média aritmética das coordenadas dos vértices.
  5. Isso não é coincidência — é a própria definição do baricentro como ponto de equilíbrio geométrico.
$$G = \left(\frac{x_A + x_B + x_C}{3}, \frac{y_A + y_B + y_C}{3}\right) = (\bar{x}, \bar{y})$$ $$G = \left(6, \frac{10}{3}\right) \approx (6, 3{,}33)$$

Reflexão de Sobrevivência: O baricentro como ponto de equilíbrio geométrico é a razão pela qual ele otimiza a distribuição de recursos. Cada comunidade contribui igualmente para determinar o centro — nenhuma domina. Na Zona Devastada, este princípio garante que a central de distribuição seja acessível de forma equitativa, evitando conflitos por recursos.
