EXPANSÃO DO PERÍMETRO

Os limites atuais do assentamento são definidos por um triângulo com vértices nas coordenadas $P(0, 0)$, $Q(12, 0)$ e $R(6, 9)$. Para expandir o perímetro, cada vértice será deslocado pelo vetor que vai do baricentro até o próprio vértice. Determine as coordenadas dos novos vértices $P'$, $Q'$ e $R'$.

(a) Calcule o baricentro $G$ do triângulo original.

(b) Para cada vértice, calcule o vetor $\vec{GV}$ e some ao vértice original para obter o novo ponto.

DICA DE SOBREVIVÊNCIA

O vetor de $G$ até $P$ é $\vec{GP} = P - G$. O novo ponto é $P' = P + \vec{GP}$. Isso equivale a $P' = P + (P - G) = 2P - G$.

ACESSO NÍVEL: SUPERVISOR
  1. Baricentro: $G = \left(\frac{0 + 12 + 6}{3}, \frac{0 + 0 + 9}{3}\right) = (6, 3)$.
  2. Vetores de $G$ até cada vértice: $\vec{GP} = P - G = (0, 0) - (6, 3) = (-6, -3)$ $\vec{GQ} = Q - G = (12, 0) - (6, 3) = (6, -3)$ $\vec{GR} = R - G = (6, 9) - (6, 3) = (0, 6)$
  3. Novos vértices (somando o vetor ao ponto original): $P' = P + \vec{GP} = (0, 0) + (-6, -3) = (-6, -3)$ $Q' = Q + \vec{GQ} = (12, 0) + (6, -3) = (18, -3)$ $R' = R + \vec{GR} = (6, 9) + (0, 6) = (6, 15)$
  4. Note que $P' = 2P - G = 2(0,0) - (6,3) = (-6, -3)$. A fórmula geral é $V' = 2V - G$.
$$G = (6, 3), \quad P' = (-6, -3), \quad Q' = (18, -3), \quad R' = (6, 15)$$ $$V' = V + \vec{GV} = V + (V - G) = 2V - G$$

Reflexão de Sobrevivência: Expandir perímetros usando o baricentro como referência é uma operação puramente algébrica com pares ordenados. Cada vértice "se afasta" do centro geométrico na mesma direção original, dobrando sua distância ao baricentro. Na Zona Devastada, esta técnica permite expandir territórios de forma simétrica e previsível.
