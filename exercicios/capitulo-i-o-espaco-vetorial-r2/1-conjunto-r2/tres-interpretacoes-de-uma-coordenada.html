TRÊS INTERPRETAÇÕES DE UMA COORDENADA

No plano cartesiano, desenhe o ponto $S(2, 3)$. Agora represente o mesmo par de números $(2, 3)$ como vetor $\vec{u} = (2, 3)$ — uma seta saindo da origem. Por fim, calcule o deslocamento do ponto $P(1, 1)$ para o ponto $Q(3, 4)$: subtraindo as coordenadas, obtemos $(3-1,\, 4-1) = (2, 3)$. Marque $P$ e $Q$, desenhe a seta de $P$ para $Q$ e compare com o vetor $\vec{u}$ na origem. As três representações — o ponto, o vetor e o deslocamento — usam o mesmo par de números, mas dizem coisas diferentes. Em suas próprias palavras, explique o que cada uma significa.

DICA DE SOBREVIVÊNCIA

O ponto $S(2, 3)$ é um local fixo. O vetor $\vec{u} = (2, 3)$ é uma seta saindo de $(0, 0)$. O deslocamento de $P$ para $Q$ é a "receita" de como chegar de um lugar a outro. Todas usam $(2, 3)$, mas representam conceitos diferentes.

ACESSO NÍVEL: SUPERVISOR
  1. O ponto $S(2, 3)$: é uma posição fixa no plano — o lugar onde se encontra um abrigo, por exemplo. É um "lugar".
  2. O vetor $\vec{u} = (2, 3)$: é uma seta saindo da origem, representando um movimento de "2 à direita, 3 para cima". É uma "direção".
  3. O deslocamento de $P(1, 1)$ para $Q(3, 4)$: é $(2, 3)$. A seta $P \to Q$ tem a mesma direção e "tamanho" que $\vec{u}$, mas sai de $P$ ao invés da origem. É um "movimento" entre dois lugares.
  4. Resumo: as mesmas coordenadas $(2, 3)$ podem ser um lugar ($S$), uma direção ($\vec{u}$) ou um movimento ($P \to Q$). O contexto determina o significado.
$$\text{Lugar: } S(2, 3) \qquad \text{Direção: } \vec{u} = (2, 3) \qquad \text{Movimento: } P(1,1) \to Q(3,4) = (2, 3)$$

Reflexão de Sobrevivência: Na Zona Devastada, "ir até $(2, 3)$" e "mover-se $(2, 3)$" são ordens completamente diferentes. A primeira diz para onde; a segunda diz como. Um radiooperador que domina a notação pode transmitir com precisão cirúrgica, enquanto um que não domina pode enviar uma equipe ao abismo. A notação não é decoreba — é código de sobrevivência.
