REPRESENTAÇÃO GRÁFICA DE PONTOS-CHAVE

Desenhe no plano cartesiano os seguintes pontos de interesse no território ao redor do Refúgio-Tec: base principal $(0, 0)$, torre de vigilância $(0, 4)$, lago contaminado $(3, -2)$, depósito de suprimentos $(2, 5)$ e acampamento temporário $(-4, -3)$. Use uma escala apropriada e indique claramente cada ponto, identificando em qual quadrante ou eixo cada um se encontra.

DICA DE SOBREVIVÊNCIA

Trace dois eixos perpendiculares se cruzando na origem $(0, 0)$, marque divisões iguais em cada eixo e posicione cada ponto caminhando o número correto de unidades nas direções horizontal e vertical apropriadas.

ACESSO NÍVEL: SUPERVISOR
  1. Trace um plano cartesiano com os eixos $x$ (horizontal) e $y$ (vertical) se cruzando na origem $(0, 0)$.
  2. Marque divisões iguais em cada eixo, por exemplo, a cada 1 unidade.
  3. Para representar a base principal $(0, 0)$, marque o ponto exatamente na origem onde os eixos se cruzam.
  4. Para a torre de vigilância $(0, 4)$, partindo da origem, ande 0 unidades na horizontal (sobre o eixo $y$) e 4 unidades para cima — este ponto está sobre o eixo $y$ positivo.
  5. Para o lago contaminado $(3, -2)$, partindo da origem, ande 3 unidades para a direita e 2 unidades para baixo — Quadrante IV.
  6. Para o depósito de suprimentos $(2, 5)$, partindo da origem, ande 2 unidades para a direita e 5 unidades para cima — Quadrante I.
  7. Para o acampamento temporário $(-4, -3)$, partindo da origem, ande 4 unidades para a esquerda e 3 unidades para baixo — Quadrante III.
$$\text{Base principal: } (0, 0) \text{ — Origem}$$ $$\text{Torre de vigilância: } (0, 4) \text{ — Eixo } y \text{ positivo}$$ $$\text{Lago contaminado: } (3, -2) \text{ — Quadrante IV}$$ $$\text{Depósito de suprimentos: } (2, 5) \text{ — Quadrante I}$$ $$\text{Acampamento temporário: } (-4, -3) \text{ — Quadrante III}$$

Reflexão de Sobrevivência: A capacidade de visualizar pontos em um plano cartesiano permite criar mapas mentais do território, essenciais quando os mapas físicos se tornaram raridade. Esta habilidade de "ver" as coordenadas pode ser a diferença entre encontrar um caminho seguro pela Zona Devastada ou ficar preso em uma área perigosa.
