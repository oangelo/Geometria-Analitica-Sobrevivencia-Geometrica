> MANUAL DE SOBREVIVÊNCIA MATEMÁTICA: PONTO MÉDIO E BARICENTRO

Após o Grande Colapso, a habilidade de calcular pontos estratégicos no plano tornou-se crucial para sobrevivência. A localização precisa de abrigos, rotas de evacuação e pontos de encontro pode significar a diferença entre vida e morte na Zona Devastada.

O ponto médio representa a localização exata entre dois pontos, permitindo encontrar locais equidistantes para distribuição de recursos ou planejamento de rotas de suprimentos. Já o baricentro, ou centro de gravidade de um triângulo, indica o ponto de equilíbrio ideal para construções resistentes ou sistemas de defesa triangulares.

Dominar estes conceitos permitirá otimizar rotas de coleta, planejar estruturas estáveis e maximizar a eficiência dos recursos escassos que restam em nosso novo mundo.

$\text{Ponto Médio entre } A(x_1, y_1) \text{ e } B(x_2, y_2):$ $$M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$$
$\text{Baricentro do triângulo } A(x_1, y_1), B(x_2, y_2), C(x_3, y_3):$ $$G = \left(\frac{x_1 + x_2 + x_3}{3}, \frac{y_1 + y_2 + y_3}{3}\right)$$

As técnicas a seguir são fundamentais para sobreviventes matemáticos:

  • Cálculo direto usando fórmulas (método rápido para situações de emergência)
  • Verificação de propriedades geométricas (quando recursos computacionais estão indisponíveis)
  • Aplicação em contextos práticos (posicionamento, divisão de território, rotas de fuga)

PONTO DE ENCONTRO SEGURO

Dois sobreviventes estão localizados nas coordenadas A(2, 3) e B(8, 5). Para minimizar o risco de exposição à radiação, eles devem se encontrar exatamente no ponto médio entre suas posições. Determine as coordenadas deste ponto de encontro seguro.

Use a fórmula do ponto médio, somando as coordenadas x e dividindo por 2, e fazendo o mesmo com as coordenadas y.

ACESSO NÍVEL: SUPERVISOR
  1. Identifique as coordenadas dos pontos dados: A(2, 3) e B(8, 5)
  2. Aplique a fórmula do ponto médio: $M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$
  3. Substitua os valores: $M = \left(\frac{2 + 8}{2}, \frac{3 + 5}{2}\right)$
  4. Simplifique: $M = \left(\frac{10}{2}, \frac{8}{2}\right) = (5, 4)$
$$M = \left(\frac{2 + 8}{2}, \frac{3 + 5}{2}\right) = (5, 4)$$

Reflexão de Sobrevivência: Calcular pontos médios permite encontrar locais equidistantes que minimizam o tempo de exposição em ambientes hostis. Essa técnica é vital para planejar pontos de encontro que sejam igualmente acessíveis a diferentes grupos de sobreviventes.

POSTO DE OBSERVAÇÃO

Três patrulhas reportaram avistamentos de mutantes nas coordenadas A(-2, 1), B(4, 7) e C(4, -5). O comandante necessita posicionar um posto de observação no baricentro destes três pontos. Calcule as coordenadas deste posto estratégico.

O baricentro é obtido calculando a média aritmética de todas as coordenadas x e todas as coordenadas y dos três pontos.

ACESSO NÍVEL: SUPERVISOR

$$G = \left(\frac{-2 + 4 + 4}{3}, \frac{1 + 7 + (-5)}{3}\right) = (2, 1)$$

Reflexão de Sobrevivência: O baricentro oferece um ponto de equilíbrio ideal para monitorar áreas triangulares. Posicionar recursos neste local estratégico permite cobertura eficiente, minimizando a necessidade de múltiplos postos em um mundo onde cada recurso é precioso.

ESTOQUES DE MEDICAMENTOS

Um assentamento tem dois depósitos de RemoveRad localizados em D(3, -1) e E(7, 5). Para facilitar o acesso durante emergências, um novo estoque deve ser construído no ponto médio entre estes depósitos. Se um sobrevivente parte deste novo ponto em direção às coordenadas (8, 0), qual será a distância total percorrida para ir do ponto médio até este destino?

Primeiro encontre o ponto médio entre D e E, depois calcule a distância desse ponto até as coordenadas (8, 0) usando a fórmula da distância entre dois pontos.

ACESSO NÍVEL: SUPERVISOR
  1. Calcule o ponto médio M entre D(3, -1) e E(7, 5): $M = \left(\frac{3 + 7}{2}, \frac{-1 + 5}{2}\right) = (5, 2)$
  2. Agora, calcule a distância do ponto M(5, 2) até o ponto P(8, 0)
  3. Use a fórmula da distância: $d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$
  4. Substitua os valores: $d = \sqrt{(8 - 5)^2 + (0 - 2)^2} = \sqrt{9 + 4} = \sqrt{13}$
$$d = \sqrt{(8 - 5)^2 + (0 - 2)^2} = \sqrt{9 + 4} = \sqrt{13}$$

Reflexão de Sobrevivência: Em tempos de escassez, a otimização de rotas é crucial. Conhecer o ponto médio entre locais importantes e calcular distâncias precisas permite planejar o uso de recursos limitados como combustível e tempo, aumentando as chances de sobrevivência durante deslocamentos perigosos.

Um artefato tecnológico importante foi encontrado nas coordenadas (5, 4). Calcule a distância do artefato até o sistema de comunicação central.

ACESSO NÍVEL: SUPERVISOR

  3. Calcule a distância entre G(3, 4) e P(5, 4): $d = \sqrt{(5 - 3)^2 + (4 - 4)^2} = \sqrt{4 + 0} = 2$

EXPANSÃO DO PERÍMETRO

Os limites atuais do assentamento são definidos por um triângulo com vértices nas coordenadas P(0, 0), Q(12, 0) e R(6, 9). Para expandir o perímetro, cada vértice será movido 2 unidades para fora a partir do baricentro, mantendo a mesma direção. Determine as coordenadas dos novos vértices P', Q' e R'.

Encontre o baricentro G e, para cada vértice, calcule o vetor que vai de G até o vértice, estendendo esse vetor para obter os novos pontos.

ACESSO NÍVEL: SUPERVISOR
  1. Calcule o baricentro G do triângulo: $G = \left(\frac{0 + 12 + 6}{3}, \frac{0 + 0 + 9}{3}\right) = (6, 3)$
  2. Para cada vértice, calculamos os vetores diretores $\overrightarrow{GP}$, $\overrightarrow{GQ}$ e $\overrightarrow{GR}$:
    $\overrightarrow{GP} = (0 - 6, 0 - 3) = (-6, -3)$
    $\overrightarrow{GQ} = (12 - 6, 0 - 3) = (6, -3)$
    $\overrightarrow{GR} = (6 - 6, 9 - 3) = (0, 6)$
  3. Normalizamos e escalamos cada vetor para determinar o deslocamento:
    $|\overrightarrow{GP}| = \sqrt{(-6)^2 + (-3)^2} = \sqrt{36 + 9} = \sqrt{45} = 3\sqrt{5}$
    $|\overrightarrow{GQ}| = \sqrt{6^2 + (-3)^2} = \sqrt{36 + 9} = \sqrt{45} = 3\sqrt{5}$
    $|\overrightarrow{GR}| = \sqrt{0^2 + 6^2} = 6$
  4. Os novos vértices são dados pelas expressões:
    $P' = P + 2 \cdot \frac{\overrightarrow{GP}}{|\overrightarrow{GP}|} = (0, 0) + 2 \cdot \frac{(-6, -3)}{3\sqrt{5}} = (0, 0) + \frac{2}{3\sqrt{5}} \cdot (-6, -3)$
    $Q' = Q + 2 \cdot \frac{\overrightarrow{GQ}}{|\overrightarrow{GQ}|} = (12, 0) + 2 \cdot \frac{(6, -3)}{3\sqrt{5}} = (12, 0) + \frac{2}{3\sqrt{5}} \cdot (6, -3)$
    $R' = R + 2 \cdot \frac{\overrightarrow{GR}}{|\overrightarrow{GR}|} = (6, 9) + 2 \cdot \frac{(0, 6)}{6} = (6, 9) + \frac{2}{6} \cdot (0, 6)$
  5. Simplificando:
    $P' = (0, 0) + \frac{2}{3\sqrt{5}} \cdot (-6, -3) = (-\frac{4}{\sqrt{5}}, -\frac{2}{\sqrt{5}}) \approx (-1.79, -0.89)$
    $Q' = (12, 0) + \frac{2}{3\sqrt{5}} \cdot (6, -3) = (12 + \frac{4}{\sqrt{5}}, 0 - \frac{2}{\sqrt{5}}) \approx (13.79, -0.89)$
    $R' = (6, 9) + \frac{1}{3} \cdot (0, 6) = (6, 9 + 2) = (6, 11)$
$$G = (6, 3)$$ $$P' \approx (-1.79, -0.89)$$ $$Q' \approx (13.79, -0.89)$$ $$R' = (6, 11)$$

Reflexão de Sobrevivência: Expandir perímetros de segurança é uma tarefa crítica em zonas hostis. A capacidade de calcular coordenadas precisas para novas fronteiras, usando o baricentro como referência, permite o crescimento organizado dos assentamentos e otimiza a distribuição de recursos de defesa.

REDE DE ABASTECIMENTO

Três comunidades estão localizadas nos pontos A(2, 1), B(10, 1) e C(6, 8). Para economizar recursos, uma central de distribuição será construída no baricentro deste triângulo. Prove que a soma das distâncias ao quadrado de cada comunidade até esta central é mínima comparada a qualquer outro ponto possível.

Para qualquer ponto P(x, y), calcule a soma das distâncias ao quadrado de P até os três pontos dados e verifique que esta soma é mínima quando P é o baricentro.

ACESSO NÍVEL: SUPERVISOR
  1. Calculamos o baricentro G do triângulo: $G = \left(\frac{2 + 10 + 6}{3}, \frac{1 + 1 + 8}{3}\right) = (6, \frac{10}{3}) \approx (6, 3.33)$
  2. Para qualquer ponto P(x, y), a soma das distâncias ao quadrado é dada por:
    $S(x, y) = |PA|^2 + |PB|^2 + |PC|^2$
    $S(x, y) = (x - 2)^2 + (y - 1)^2 + (x - 10)^2 + (y - 1)^2 + (x - 6)^2 + (y - 8)^2$
  3. Expandindo: $S(x, y) = 3x^2 - 36x + 144 + 3y^2 - 20y + 66$
  4. Para encontrar o mínimo, derivamos e igualamos a zero:
    $\frac{\partial S}{\partial x} = 6x - 36 = 0 \implies x = 6$
    $\frac{\partial S}{\partial y} = 6y - 20 = 0 \implies y = \frac{10}{3}$
  5. O ponto (6, \frac{10}{3}) é exatamente o baricentro G, provando que a soma das distâncias ao quadrado é mínima neste ponto.
$$S(x, y) = |PA|^2 + |PB|^2 + |PC|^2$$ $$\frac{\partial S}{\partial x} = 0 \implies x = 6$$ $$\frac{\partial S}{\partial y} = 0 \implies y = \frac{10}{3}$$

Reflexão de Sobrevivência: O posicionamento otimizado de centros de recursos é crucial em tempos onde cada grama de combustível e cada minuto de exposição contam. O baricentro oferece a solução matemática ideal para minimizar o custo total de transporte em uma rede de comunidades, maximizando a eficiência e prolongando a sobrevivência.

O PONTO DE EQUILÍBRIO

As coordenadas A(0, 0), B(8, 0) e C(4, 4) representam três pontos de extração de água. Um sistema de canalização deve conectar estes pontos, com uma válvula central posicionada no ponto médio do segmento que liga o ponto C ao ponto médio de AB. Determine as coordenadas da válvula e a quantidade total de tubulação necessária para conectar todos os pontos até a válvula.

Este é um problema de aplicação sequencial de ponto médio.

ACESSO NÍVEL: SUPERVISOR
  1. Calcule o ponto médio M de A(0, 0) e B(8, 0): $M = \left(\frac{0 + 8}{2}, \frac{0 + 0}{2}\right) = (4, 0)$
  2. Agora, calcule o ponto médio V entre M(4, 0) e C(4, 4): $V = \left(\frac{4 + 4}{2}, \frac{0 + 4}{2}\right) = (4, 2)$
  3. Para calcular a quantidade total de tubulação, determine as distâncias de cada ponto até a válvula:
    $|AV| = \sqrt{(4 - 0)^2 + (2 - 0)^2} = \sqrt{16 + 4} = \sqrt{20} = 2\sqrt{5}$
    $|BV| = \sqrt{(4 - 8)^2 + (2 - 0)^2} = \sqrt{16 + 4} = \sqrt{20} = 2\sqrt{5}$
    $|CV| = \sqrt{(4 - 4)^2 + (2 - 4)^2} = \sqrt{0 + 4} = 2$
  4. A quantidade total de tubulação será: $|AV| + |BV| + |CV| = 2\sqrt{5} + 2\sqrt{5} + 2 = 4\sqrt{5} + 2$
$$V = (4, 2)$$ $$\text{Tubulação total} = 4\sqrt{5} + 2 \approx 10.94 \text{ unidades}$$

Reflexão de Sobrevivência: Em infraestrutura crítica como sistemas hídricos, a aplicação iterativa do conceito de ponto médio permite criar nós de distribuição que minimizam o uso de materiais escassos. Esta otimização não é apenas questão de economia, mas de viabilidade em um mundo onde cada recurso salvo pode significar mais vidas preservadas.

A TRINDADE PROTETORA

Três torres de vigia formam um triângulo nas coordenadas A(1, 1), B(7, 1) e C(4, 5√3). Um novo sistema de triagem deve ser posicionado no baricentro. Se cada torre tem um gerador que pode fornecer energia proporcional ao inverso da distância ao quadrado, determine o ponto X na região triangular onde a soma das contribuições energéticas é máxima. Este ponto coincide com o baricentro?

A contribuição energética de cada torre para um ponto X segue a lei do inverso do quadrado da distância.

ACESSO NÍVEL: SUPERVISOR
  1. Primeiro, calculamos o baricentro G: $G = \left(\frac{1 + 7 + 4}{3}, \frac{1 + 1 + 5\sqrt{3}}{3}\right) = (4, \frac{2 + 5\sqrt{3}}{3})$
  2. Para a contribuição energética, definimos uma função: $E(x, y) = \frac{1}{|XA|^2} + \frac{1}{|XB|^2} + \frac{1}{|XC|^2}$
  3. Expandindo: $E(x, y) = \frac{1}{(x-1)^2 + (y-1)^2} + \frac{1}{(x-7)^2 + (y-1)^2} + \frac{1}{(x-4)^2 + (y-5\sqrt{3})^2}$
  4. Analisando as propriedades deste triângulo, percebemos que é um triângulo equilátero (podemos verificar as distâncias entre os vértices)
  5. Para um triângulo equilátero, o ponto que maximiza a soma dos inversos das distâncias ao quadrado não é o baricentro, mas o ponto de Fermat (ou ponto de Torricelli), que coincide com o baricentro apenas para triângulos equiláteros
  6. Como este é um triângulo equilátero (AB = BC = AC = 6), o ponto X que maximiza a energia coincide com o baricentro G
$$G = \left(4, \frac{2 + 5\sqrt{3}}{3}\right)$$ $$X = G = \left(4, \frac{2 + 5\sqrt{3}}{3}\right)$$

Reflexão de Sobrevivência: A distribuição ideal de recursos energéticos em um sistema triangular demonstra um princípio fundamental de sobrevivência: geometria e física trabalham juntas na otimização. Conhecer propriedades especiais de configurações geométricas permite criar redes resilientes que aproveitam ao máximo recursos limitados.

ROTA DE EVACUAÇÃO SEGMENTADA

Três pontos de controle estão situados em A(0, 0), B(10, 0) e C(5, 8). Uma rota de evacuação deve ser planejada saindo do ponto A, passando pelo ponto médio M1 entre A e B, depois pelo baricentro G do triângulo ABC, em seguida pelo ponto médio M2 entre B e C, e finalmente chegando em C. Determine o comprimento total desta rota e compare com o perímetro do triângulo ABC.

Calcule cada segmento da rota sequencialmente e some as distâncias.

ACESSO NÍVEL: SUPERVISOR
  1. Calcule o ponto médio M1 entre A(0, 0) e B(10, 0): $M1 = (5, 0)$
  2. Calcule o baricentro G do triângulo: $G = \left(\frac{0 + 10 + 5}{3}, \frac{0 + 0 + 8}{3}\right) = (5, \frac{8}{3}) \approx (5, 2.67)$
  3. Calcule o ponto médio M2 entre B(10, 0) e C(5, 8): $M2 = \left(\frac{10 + 5}{2}, \frac{0 + 8}{2}\right) = (7.5, 4)$
  4. Calcule o comprimento de cada segmento da rota:
    $|AM1| = \sqrt{(5 - 0)^2 + (0 - 0)^2} = 5$
    $|M1G| = \sqrt{(5 - 5)^2 + (2.67 - 0)^2} = 2.67$
    $|GM2| = \sqrt{(7.5 - 5)^2 + (4 - 2.67)^2} = \sqrt{6.25 + 1.78} \approx 2.94$
    $|M2C| = \sqrt{(5 - 7.5)^2 + (8 - 4)^2} = \sqrt{6.25 + 16} = \sqrt{22.25} \approx 4.72$
  5. Comprimento total da rota: $5 + 2.67 + 2.94 + 4.72 = 15.33$
  6. Agora, calculamos o perímetro do triângulo ABC:
    $|AB| = \sqrt{(10 - 0)^2 + (0 - 0)^2} = 10$
    $|BC| = \sqrt{(5 - 10)^2 + (8 - 0)^2} = \sqrt{25 + 64} = \sqrt{89} \approx 9.43$
    $|CA| = \sqrt{(0 - 5)^2 + (0 - 8)^2} = \sqrt{25 + 64} = \sqrt{89} \approx 9.43$
  7. Perímetro do triângulo: $10 + 9.43 + 9.43 = 28.86$
$$\text{Comprimento da rota} = |AM1| + |M1G| + |GM2| + |M2C| \approx 15.33$$ $$\text{Perímetro do triângulo} = |AB| + |BC| + |CA| \approx 28.86$$ $$\text{Razão} = \frac{15.33}{28.86} \approx 0.53 \text{ ou } 53\%$$

Reflexão de Sobrevivência: Rotas que utilizam pontos médios e baricentros estratégicos podem reduzir significativamente as distâncias percorridas, economizando até 47% dos recursos de deslocamento em comparação com rotas perimetrais. Em um mundo hostil, esta eficiência pode ser a diferença entre alcançar o abrigo antes do anoitecer ou enfrentar os perigos noturnos da Zona Devastada.

A CONVERGÊNCIA DAS MEIAS-VIDAS

Quatro estações de monitoramento formam um quadrilátero com vértices A(0, 0), B(8, 0), C(9, 7) e D(1, 7). Para criar uma rede de comunicação eficiente, planeja-se unir os pontos médios dos lados consecutivos, formando um novo quadrilátero. Prove que este novo quadrilátero é sempre um paralelogramo, independentemente da forma do quadrilátero original, e calcule o perímetro deste novo quadrilátero.

ACESSO NÍVEL: SUPERVISOR
  1. Calculamos os pontos médios dos lados do quadrilátero:
    $M_{AB} = \left(\frac{0 + 8}{2}, \frac{0 + 0}{2}\right) = (4, 0)$
    $M_{BC} = \left(\frac{8 + 9}{2}, \frac{0 + 7}{2}\right) = (8.5, 3.5)$
    $M_{CD} = \left(\frac{9 + 1}{2}, \frac{7 + 7}{2}\right) = (5, 7)$
    $M_{DA} = \left(\frac{1 + 0}{2}, \frac{7 + 0}{2}\right) = (0.5, 3.5)$
  2. Para provar que o quadrilátero $M_{AB}M_{BC}M_{CD}M_{DA}$ é um paralelogramo, verificamos se os lados opostos são paralelos e iguais.
    $\overrightarrow{M_{AB}M_{BC}} = (8.5 - 4, 3.5 - 0) = (4.5, 3.5)$
    $\overrightarrow{M_{CD}M_{DA}} = (0.5 - 5, 3.5 - 7) = (-4.5, -3.5)$
  3. Verificamos que $\overrightarrow{M_{AB}M_{BC}} = -\overrightarrow{M_{CD}M_{DA}}$, o que prova que os lados opostos são paralelos e de mesmo comprimento
  4. Da mesma forma, podemos verificar para os outros dois lados:
    $\overrightarrow{M_{BC}M_{CD}} = (5 - 8.5, 7 - 3.5) = (-3.5, 3.5)$
    $\overrightarrow{M_{DA}M_{AB}} = (4 - 0.5, 0 - 3.5) = (3.5, -3.5)$
  5. Novamente, $\overrightarrow{M_{BC}M_{CD}} = -\overrightarrow{M_{DA}M_{AB}}$
  6. Isso prova matematicamente que o quadrilátero formado pelos pontos médios é sempre um paralelogramo
  7. Para calcular o perímetro, precisamos da soma dos comprimentos dos lados:
    $|M_{AB}M_{BC}| = \sqrt{4.5^2 + 3.5^2} = \sqrt{20.25 + 12.25} = \sqrt{32.5} \approx 5.7$
    $|M_{BC}M_{CD}| = \sqrt{(-3.5)^2 + 3.5^2} = \sqrt{24.5} \approx 4.95$
    $|M_{CD}M_{DA}| = \sqrt{(-4.5)^2 + (-3.5)^2} = \sqrt{32.5} \approx 5.7$
    $|M_{DA}M_{AB}| = \sqrt{3.5^2 + (-3.5)^2} = \sqrt{24.5} \approx 4.95$
  8. Perímetro = $2 \cdot (5.7 + 4.95) = 21.3$
$$\overrightarrow{M_{AB}M_{BC}} = -\overrightarrow{M_{CD}M_{DA}}$$ $$\overrightarrow{M_{BC}M_{CD}} = -\overrightarrow{M_{DA}M_{AB}}$$ $$\text{Perímetro do paralelogramo} \approx 21.3 \text{ unidades}$$

Reflexão de Sobrevivência: A descoberta de que pontos médios de quadriláteros sempre formam paralelogramos garante estabilidade estrutural previsível, mesmo em terrenos irregulares. Esta propriedade invariante é valiosa para engenheiros de sobrevivência construindo estruturas defensivas com recursos limitados, permitindo projetar sistemas estáveis independentemente das condições iniciais encontradas.

A HERANÇA DO CENTRO

Um triângulo inicial T₀ tem vértices A₀(0, 0), B₀(6, 0) e C₀(3, 3√3). Definimos um processo recursivo onde, a cada iteração n, criamos um novo triângulo Tₙ utilizando os baricentros de três triângulos: o primeiro formado pelos pontos médios dos lados de Tₙ₋₁, o segundo formado pelos vértices de Tₙ₋₁, e o terceiro formado pelas medianas de Tₙ₋₁. Determine as coordenadas dos vértices de T₂ e prove que o baricentro permanece invariante através das iterações.

ACESSO NÍVEL: SUPERVISOR
  1. Primeiramente, calculamos o baricentro G₀ do triângulo T₀:
    $G_0 = \left(\frac{0 + 6 + 3}{3}, \frac{0 + 0 + 3\sqrt{3}}{3}\right) = (3, \sqrt{3})$
  2. Calculamos os pontos médios dos lados de T₀:
    $M_{AB} = \left(\frac{0 + 6}{2}, \frac{0 + 0}{2}\right) = (3, 0)$
    $M_{BC} = \left(\frac{6 + 3}{2}, \frac{0 + 3\sqrt{3}}{2}\right) = (4.5, 1.5\sqrt{3})$
    $M_{CA} = \left(\frac{3 + 0}{2}, \frac{3\sqrt{3} + 0}{2}\right) = (1.5, 1.5\sqrt{3})$
  3. O triângulo formado pelos pontos médios é T' = △(M_{AB}, M_{BC}, M_{CA})
  4. Calculamos o baricentro G' de T':
    $G' = \left(\frac{3 + 4.5 + 1.5}{3}, \frac{0 + 1.5\sqrt{3} + 1.5\sqrt{3}}{3}\right) = (3, \sqrt{3})$
  5. As medianas de T₀ unem cada vértice ao ponto médio do lado oposto, formando o triângulo T'' = △(A₀M_{BC}, B₀M_{CA}, C₀M_{AB})
  6. O baricentro G'' deste triângulo também é G'' = (3, √3)
  7. Para T₁, usamos os três baricentros calculados (que são todos iguais a (3, √3))
  8. Como todos os baricentros coincidem em (3, √3), os vértices de T₁ também serão todos (3, √3)
  9. Analogamente, todos os baricentros de T₁ serão (3, √3)
  10. Portanto, T₂ terá todos os vértices em (3, √3)
  11. Concluímos que o baricentro permanece invariante porque cada nova iteração gera triângulos cujos baricentros coincidem com o baricentro original
$$G_0 = G' = G'' = (3, \sqrt{3})$$ $$\text{Vértices de } T_2 = \{(3, \sqrt{3}), (3, \sqrt{3}), (3, \sqrt{3})\}$$

Reflexão de Sobrevivência: Em sistemas complexos com múltiplas camadas de recursividade, a descoberta de invariantes geométricos oferece pontos de estabilidade fundamentais. No caos da Zona Devastada, identificar estes "pontos de ancoragem" matemáticos permite projetar sistemas recursivos de recursos que mantêm sua eficiência mesmo após múltiplas reorganizações, garantindo resiliência a longo prazo.

O ÚLTIMO REFÚGIO

Um drone de reconhecimento identificou cinco pontos estratégicos nas coordenadas A(0, 0), B(8, 0), C(8, 6), D(4, 8) e E(0, 6). Para maximizar a segurança, um abrigo deve ser construído em um local X tal que a soma dos quadrados das distâncias deste ponto até os cinco pontos estratégicos seja mínima. Determine as coordenadas deste ponto e prove que coincide com a média aritmética das coordenadas dos cinco pontos originais. Qual seria a distância média do abrigo até os pontos estratégicos?

ACESSO NÍVEL: SUPERVISOR
  1. Consideramos um ponto genérico X(x, y) e definimos a função:
    $f(x, y) = |XA|^2 + |XB|^2 + |XC|^2 + |XD|^2 + |XE|^2$
  2. Substituindo as coordenadas:
    $f(x, y) = (x-0)^2 + (y-0)^2 + (x-8)^2 + (y-0)^2 + (x-8)^2 + (y-6)^2 + (x-4)^2 + (y-8)^2 + (x-0)^2 + (y-6)^2$
  3. Agrupando os termos:
    $f(x, y) = 5x^2 - 2(0+8+8+4+0)x + (0^2+8^2+8^2+4^2+0^2) + 5y^2 - 2(0+0+6+8+6)y + (0^2+0^2+6^2+8^2+6^2)$
    $f(x, y) = 5x^2 - 40x + 5y^2 - 40y + C$, onde $C$ é uma constante
  4. Para encontrar o mínimo, derivamos parcialmente e igualamos a zero:
    $\frac{\partial f}{\partial x} = 10x - 40 = 0 \implies x = 4$
    $\frac{\partial f}{\partial y} = 10y - 40 = 0 \implies y = 4$
  5. Verificamos que este ponto é um mínimo calculando a matriz Hessiana:
    $\frac{\partial^2 f}{\partial x^2} = 10 > 0$
    $\frac{\partial^2 f}{\partial y^2} = 10 > 0$
    $\frac{\partial^2 f}{\partial x \partial y} = 0$
    $\det(H) = 10 \cdot 10 - 0 \cdot 0 = 100 > 0$
  6. O ponto (4, 4) é de fato um mínimo global
  7. Calculamos a média aritmética das coordenadas dos cinco pontos:
    $\bar{x} = \frac{0 + 8 + 8 + 4 + 0}{5} = \frac{20}{5} = 4$
    $\bar{y} = \frac{0 + 0 + 6 + 8 + 6}{5} = \frac{20}{5} = 4$
    $\bar{P} = (4, 4)$
  8. Isto prova que o ponto que minimiza a soma dos quadrados das distâncias é a média aritmética das coordenadas
  9. Calculamos a distância de X até cada ponto:
    $|XA| = \sqrt{(4-0)^2 + (4-0)^2} = \sqrt{32} = 4\sqrt{2}$
    $|XB| = \sqrt{(4-8)^2 + (4-0)^2} = \sqrt{32} = 4\sqrt{2}$
    $|XC| = \sqrt{(4-8)^2 + (4-6)^2} = \sqrt{20} = 2\sqrt{5}$
    $|XD| = \sqrt{(4-4)^2 + (4-8)^2} = 4$
    $|XE| = \sqrt{(4-0)^2 + (4-6)^2} = \sqrt{20} = 2\sqrt{5}$
  10. A distância média é:
    $\text{Distância média} = \frac{|XA| + |XB| + |XC| + |XD| + |XE|}{5} = \frac{4\sqrt{2} + 4\sqrt{2} + 2\sqrt{5} + 4 + 2\sqrt{5}}{5} = \frac{8\sqrt{2} + 4\sqrt{5} + 4}{5} \approx 4.33$
$$X = \left(\frac{x_1 + x_2 + ... + x_n}{n}, \frac{y_1 + y_2 + ... + y_n}{n}\right) = (4, 4)$$ $$\text{Distância média} \approx 4.33 \text{ unidades}$$

Reflexão de Sobrevivência: A prova matemática de que o ponto de distância quadrática mínima é precisamente a média das coordenadas tem implicações críticas para planejamento de abrigos: sistemas complexos com múltiplos pontos de interesse encontram seu equilíbrio ótimo através de uma simples média aritmética. Na escassez pós-apocalíptica, este princípio permite otimizar rapidamente posicionamentos críticos sem cálculos complexos, salvando vidas quando cada segundo conta.
