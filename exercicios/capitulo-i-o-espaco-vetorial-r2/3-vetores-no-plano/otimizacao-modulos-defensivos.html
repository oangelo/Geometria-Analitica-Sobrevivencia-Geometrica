OTIMIZAÇÃO DE MÓDULOS DEFENSIVOS

O Refúgio-Tec possui dois tipos de módulos de barreira. O tipo $A$ produz um vetor de defesa $\vec{a} = (2, 1)$ e o tipo $B$ produz $\vec{b} = (1, 3)$. Para a proteção completa do perímetro, os engenheiros calcularam que o vetor resultante precisa ser $\vec{r} = (7, 11)$. Determine quantos módulos de cada tipo ($x$ do tipo $A$ e $y$ do tipo $B$) são necessários, resolvendo o sistema que surge da equação $x\vec{a} + y\vec{b} = \vec{r}$.

DICA DE SOBREVIVÊNCIA

A equação $x\vec{a} + y\vec{b} = \vec{r}$ gera um sistema de duas equações: uma para a componente $x$ e outra para a componente $y$. Use o método de eliminação ou substituição para resolver.

ACESSO NÍVEL: SUPERVISOR
  1. Montamos a equação: $x(2, 1) + y(1, 3) = (7, 11)$.
  2. Isso gera o sistema: componente $x$: $2x + y = 7$; componente $y$: $x + 3y = 11$.
  3. Da primeira equação: $y = 7 - 2x$.
  4. Substituindo na segunda: $x + 3(7 - 2x) = 11 \implies x + 21 - 6x = 11 \implies -5x = -10 \implies x = 2$.
  5. Então $y = 7 - 2(2) = 3$.
  6. Verificação: $2(2, 1) + 3(1, 3) = (4, 2) + (3, 9) = (7, 11)$. Confere.
$$\begin{cases} 2x + y = 7 \\ x + 3y = 11 \end{cases} \implies x = 2,\ y = 3$$ $$2\vec{a} + 3\vec{b} = 2(2, 1) + 3(1, 3) = (7, 11) = \vec{r}$$

Reflexão de Sobrevivência: Cada módulo defensivo custa recursos escassos. Usar a álgebra vetorial para calcular a combinação exata evita desperdício: 2 módulos $A$ e 3 módulos $B$ — nem um a mais, nem um a menos. Na Zona Devastada, otimização não é luxo, é sobrevivência.
