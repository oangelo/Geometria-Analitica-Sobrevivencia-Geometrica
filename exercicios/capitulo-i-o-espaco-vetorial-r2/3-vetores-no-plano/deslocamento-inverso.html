DESLOCAMENTO INVERSO

Dois postos de vigilância estão conectados pelo vetor $\overrightarrow{PQ} = (5, -3)$, que descreve como ir de $P$ até $Q$. O SUPERVISOR precisa transmitir para a guarnição em $Q$ como retornar a $P$. O vetor de retorno é o oposto de $\overrightarrow{PQ}$ — basta inverter o sinal de cada componente. Calcule $-\overrightarrow{PQ}$, verifique que $\overrightarrow{PQ} + (-\overrightarrow{PQ}) = \vec{0}$, e desenhe os dois vetores: o original de $P$ para $Q$ e o inverso de $Q$ para $P$.

DICA DE SOBREVIVÊNCIA

O vetor oposto de $(a, b)$ é $(-a, -b)$. Para verificar, some os dois: $(a, b) + (-a, -b) = (0, 0)$. No desenho, o vetor oposto tem mesmo comprimento mas sentido contrário.

ACESSO NÍVEL: SUPERVISOR
  1. O vetor oposto: $-\overrightarrow{PQ} = -(5, -3) = (-5, 3)$.
  2. Verificação: $\overrightarrow{PQ} + (-\overrightarrow{PQ}) = (5, -3) + (-5, 3) = (0, 0) = \vec{0}$. Confere.
  3. Interpretação: para ir de $Q$ de volta a $P$, a guarnição deve percorrer $(-5, 3)$ — 5 unidades a oeste e 3 ao norte.
  4. No esboço, $\overrightarrow{PQ}$ é uma seta de $P$ para $Q$, e $-\overrightarrow{PQ}$ é a seta de $Q$ para $P$. Mesma reta, mesmo comprimento, sentido oposto.
$$-\overrightarrow{PQ} = -(5, -3) = (-5, 3)$$ $$\overrightarrow{PQ} + (-\overrightarrow{PQ}) = \vec{0}$$ $$\text{Todo vetor } \vec{v} \text{ possui um oposto } -\vec{v} \text{ tal que } \vec{v} + (-\vec{v}) = \vec{0}$$

Reflexão de Sobrevivência: O vetor oposto é o "botão de desfazer" da navegação. Se a guarnição cometer um erro de rota, basta aplicar $-\vec{v}$ para retornar ao ponto anterior. Em uma situação de emergência, saber o vetor inverso pode ser a diferença entre chegar ao abrigo ou se perder na Zona Devastada.
