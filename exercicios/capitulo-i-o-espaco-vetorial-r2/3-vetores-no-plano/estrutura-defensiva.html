ESTRUTURA DEFENSIVA

A Refúgio-Tec está desenvolvendo uma estrutura defensiva hexagonal contra mutantes. O projeto posiciona seis torres de vigia nos vértices de um hexágono regular com centro na origem $O(0,0)$. As coordenadas dos vértices são: $A(5, 0)$, $B(2{,}5, 4{,}33)$, $C(-2{,}5, 4{,}33)$, $D(-5, 0)$, $E(-2{,}5, -4{,}33)$ e $F(2{,}5, -4{,}33)$. Calcule a soma vetorial $\vec{OA} + \vec{OB} + \vec{OC} + \vec{OD} + \vec{OE} + \vec{OF}$ e explique o significado do resultado para o design da estrutura.

DICA DE SOBREVIVÊNCIA

Some as componentes $x$ de todos os vetores e as componentes $y$ de todos os vetores separadamente. Observe os padrões de simetria — os valores positivos e negativos podem se cancelar.

ACESSO NÍVEL: SUPERVISOR
  1. Os vetores de posição são iguais às coordenadas dos vértices, pois a origem é $(0,0)$.
  2. Componente $x$ da soma: $5 + 2{,}5 + (-2{,}5) + (-5) + (-2{,}5) + 2{,}5 = 0$
  3. Componente $y$ da soma: $0 + 4{,}33 + 4{,}33 + 0 + (-4{,}33) + (-4{,}33) = 0$
  4. A soma vetorial é $\vec{0} = (0, 0)$.
  5. Isso ocorre porque os vértices estão simetricamente distribuídos: cada vértice no "lado direito" tem um correspondente no "lado esquerdo" com componente $x$ oposta, e o mesmo vale para $y$.
$$\sum_{i=1}^{6} \vec{OV_i} = \vec{0}$$ $$\text{Componente } x\text{: } 5 + 2{,}5 - 2{,}5 - 5 - 2{,}5 + 2{,}5 = 0$$ $$\text{Componente } y\text{: } 0 + 4{,}33 + 4{,}33 + 0 - 4{,}33 - 4{,}33 = 0$$

Reflexão de Sobrevivência: O resultado da soma vetorial sendo o vetor nulo demonstra o equilíbrio perfeito desta estrutura defensiva. As forças estruturais se cancelam mutuamente, proporcionando estabilidade máxima. Além disso, o sistema de vigilância tem cobertura uniforme em todas as direções — não há direção preferencial de ataque, tornando a estrutura excepcionalmente resistente a incursões de qualquer ângulo.
