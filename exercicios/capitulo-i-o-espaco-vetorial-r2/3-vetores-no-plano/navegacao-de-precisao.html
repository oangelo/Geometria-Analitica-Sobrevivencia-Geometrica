NAVEGAÇÃO DE PRECISÃO

Uma expedição científica precisa coletar amostras em três pontos de uma região radioativa. Partindo do abrigo $O(0,0)$, a equipe deve visitar os pontos $A(2,5)$, $B(6,3)$ e $C(4,8)$ exatamente uma vez cada e retornar a $O$. Calcule os vetores de deslocamento para cada uma das rotas a seguir e determine qual delas resulta no menor deslocamento total (soma dos vetores):

(a) Rota 1: $O \to A \to B \to C \to O$

(b) Rota 2: $O \to A \to C \to B \to O$

DICA DE SOBREVIVÊNCIA

Para cada trecho, calcule o vetor de deslocamento subtraindo o ponto de partida do ponto de chegada. Depois some todos os vetores de cada rota. Lembre-se: a soma de todos os deslocamentos de um circuito fechado sempre resulta no vetor nulo.

ACESSO NÍVEL: SUPERVISOR
  1. Rota 1 — vetores de deslocamento: $\vec{OA} = (2,5) - (0,0) = (2, 5)$ $\vec{AB} = (6,3) - (2,5) = (4, -2)$ $\vec{BC} = (4,8) - (6,3) = (-2, 5)$ $\vec{CO} = (0,0) - (4,8) = (-4, -8)$
  2. Soma dos vetores da Rota 1: $(2, 5) + (4, -2) + (-2, 5) + (-4, -8) = (0, 0) = \vec{0}$
  3. Rota 2 — vetores de deslocamento: $\vec{OA} = (2, 5)$ $\vec{AC} = (4,8) - (2,5) = (2, 3)$ $\vec{CB} = (6,3) - (4,8) = (2, -5)$ $\vec{BO} = (0,0) - (6,3) = (-6, -3)$
  4. Soma dos vetores da Rota 2: $(2, 5) + (2, 3) + (2, -5) + (-6, -3) = (0, 0) = \vec{0}$
  5. Ambas as rotas resultam no vetor nulo, o que é esperado: todo circuito fechado retorna ao ponto de partida, então o deslocamento total é sempre zero.
  6. A diferença entre as rotas não está no deslocamento final, mas nos deslocamentos intermediários — informação relevante para avaliar exposição acumulada à radiação.
$$\text{Rota 1: } (2, 5) + (4, -2) + (-2, 5) + (-4, -8) = \vec{0}$$ $$\text{Rota 2: } (2, 5) + (2, 3) + (2, -5) + (-6, -3) = \vec{0}$$ $$\text{Todo circuito fechado: } \sum \vec{d_i} = \vec{0}$$

Reflexão de Sobrevivência: O fato de toda rota fechada resultar no vetor nulo é uma propriedade fundamental dos vetores. Na prática, isso significa que ambas as rotas trazem a equipe de volta ao mesmo ponto. Porém, os deslocamentos intermediários são diferentes — conhecer cada trecho permite planejar pontos de descanso e estimar exposição acumulada, fatores que podem ser mais decisivos que o destino final.
