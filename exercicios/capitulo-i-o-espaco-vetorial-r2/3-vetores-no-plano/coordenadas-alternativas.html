COORDENADAS ALTERNATIVAS

Em uma expedição à antiga base militar, você encontrou um mapa que usa um sistema de coordenadas deslocado em relação ao seu sistema padrão. O deslocamento é de 3 unidades para leste e 2 unidades para norte. Um ponto estratégico está marcado como $(4, 2)$ neste sistema de coordenadas deslocado. Determine as coordenadas deste ponto no seu sistema padrão para programar seu dispositivo de navegação.

(a) Se o sistema deslocado tem sua origem em $(3, 2)$ do sistema padrão, como as coordenadas $(4, 2)$ do sistema deslocado se traduzem no sistema padrão?

(b) Geralize: se um ponto tem coordenadas $(x', y')$ no sistema deslocado com origem em $(a, b)$, quais são suas coordenadas $(x, y)$ no sistema padrão?

DICA DE SOBREVIVÊNCIA

Se a origem do sistema deslocado está em $(a, b)$ no sistema padrão, então um ponto $(x', y')$ no sistema deslocado corresponde ao ponto $(x' + a, y' + b)$ no sistema padrão. É como somar dois pares ordenados.

ACESSO NÍVEL: SUPERVISOR
  1. O sistema deslocado tem origem em $(3, 2)$ do sistema padrão. O ponto $(4, 2)$ no sistema deslocado está 4 unidades na horizontal e 2 unidades na vertical a partir dessa origem.
  2. Convertendo: $x = 4 + 3 = 7$ e $y = 2 + 2 = 4$. Portanto, o ponto no sistema padrão é $(7, 4)$.
  3. Generalização: para um ponto $(x', y')$ no sistema deslocado com origem em $(a, b)$, as coordenadas no sistema padrão são $(x' + a, y' + b)$.
  4. Isso equivale a somar o par ordenado $(x', y')$ ao par ordenado $(a, b)$: $(x, y) = (x', y') + (a, b)$.
$$(x, y) = (x', y') + (a, b) = (x' + a, y' + b)$$ $$(4, 2) + (3, 2) = (7, 4)$$

Reflexão de Sobrevivência: Mapas e sistemas de coordenadas de diferentes épocas ou origens são comuns em expedições. A capacidade de converter entre sistemas usando simples adição de pares ordenados é essencial para interpretar corretamente locais estratégicos, evitando erros que poderiam levar uma equipe diretamente para uma área perigosa.
