ESCUDOS VETORIAIS

O laboratório de pesquisa da Refúgio-Tec desenvolveu um sistema de escudos vetoriais contra radiação. Cada escudo projeta um campo de proteção representado por um vetor no plano. A eficácia da proteção depende de como esses campos se combinam. Três escudos são posicionados nos pontos $A(2,1)$, $B(5,3)$ e $C(3,6)$, projetando campos de proteção proporcionais aos vetores $\vec{AB}$, $\vec{BC}$ e $\vec{CA}$. Determine os três vetores de campo e calcule o vetor resultante da soma dos três campos. O que o vetor resultante revela sobre o sistema de proteção?

DICA DE SOBREVIVÊNCIA

Calcule cada vetor usando subtração de pares ordenados: $\vec{AB} = B - A$. Depois some os três vetores componente a componente.

ACESSO NÍVEL: SUPERVISOR
  1. Calculamos os três vetores de campo: $\vec{AB} = B - A = (5,3) - (2,1) = (3, 2)$ $\vec{BC} = C - B = (3,6) - (5,3) = (-2, 3)$ $\vec{CA} = A - C = (2,1) - (3,6) = (-1, -5)$
  2. Somamos os três vetores: $\vec{R} = \vec{AB} + \vec{BC} + \vec{CA} = (3, 2) + (-2, 3) + (-1, -5)$
  3. Componente $x$: $3 + (-2) + (-1) = 0$
  4. Componente $y$: $2 + 3 + (-5) = 0$
  5. O vetor resultante é $\vec{R} = (0, 0) = \vec{0}$.
  6. O vetor nulo indica que os três campos se cancelam perfeitamente — o sistema está em equilíbrio.
$$\vec{AB} = (3, 2), \quad \vec{BC} = (-2, 3), \quad \vec{CA} = (-1, -5)$$ $$\vec{R} = \vec{AB} + \vec{BC} + \vec{CA} = \vec{0}$$

Reflexão de Sobrevivência: Quando a soma dos vetores de campo resulta no vetor nulo, o sistema de escudos está em equilíbrio perfeito — não há direção preferencial de vazamento de radiação. Esta configuração triangular garante proteção uniforme em todas as direções. Em contraste, se o vetor resultante fosse não nulo, haveria uma "direção de vazamento" que precisaria ser reforçada com escudos adicionais.
