CIRCUITO FECHADO

Uma patrulha sai do abrigo $A$, segue até o posto $B$, depois até a torre de vigia $C$, e finalmente retorna a $A$. Os deslocamentos são $\overrightarrow{AB} = (3, 2)$, $\overrightarrow{BC} = (-2, 3)$ e $\overrightarrow{CA} = (-1, -5)$. Calcule a soma $\overrightarrow{AB} + \overrightarrow{BC} + \overrightarrow{CA}$ e verifique se ela é o vetor nulo $\vec{0} = (0, 0)$. O que isso revela sobre o circuito?

DICA DE SOBREVIVÊNCIA

Some os três vetores componente a componente. Se o resultado for $(0, 0)$, o deslocamento líquido é zero — a patrulha voltou ao ponto de partida. Isso é uma propriedade de todo circuito fechado.

ACESSO NÍVEL: SUPERVISOR
  1. Somamos componente a componente: componente $x$: $3 + (-2) + (-1) = 0$. Componente $y$: $2 + 3 + (-5) = 0$.
  2. A soma é $\vec{0} = (0, 0)$, o vetor nulo.
  3. Isso confirma que o circuito é fechado: a patrulha sai de $A$ e retorna a $A$. O deslocamento líquido total é zero — não há "sobra" de movimento.
  4. Propriedade geral: para qualquer circuito fechado $P_1 \to P_2 \to \cdots \to P_n \to P_1$, a soma dos vetores de deslocamento é sempre $\vec{0}$.
$$\overrightarrow{AB} + \overrightarrow{BC} + \overrightarrow{CA} = (3, 2) + (-2, 3) + (-1, -5) = (0, 0) = \vec{0}$$ $$\text{Circuito fechado} \implies \sum \overrightarrow{d_i} = \vec{0}$$

Reflexão de Sobrevivência: O vetor nulo é o selo de aprovação de um circuito bem planejado. Se a soma não for $\vec{0}$, a patrulha não voltou ao abrigo — e na Zona Devastada, passar a noite fora do abrigo é um convite para problemas com mutantes.
