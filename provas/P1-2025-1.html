Questão 1: Determinação do Circuncentro (2,0 pts)

Sejam os pontos $A = (1, 2)$, $B = (5, 4)$ e $C = (3, 8)$ no plano cartesiano.

a) (0,3 pt) Calcule os pontos médios $M_{AB}$, $M_{BC}$ e $M_{AC}$.

Gabarito

$M_{AB} = \left(\frac{x_A + x_B}{2}, \frac{y_A + y_B}{2}\right) = \left(\frac{1+5}{2}, \frac{2+4}{2}\right) = \left(\frac{6}{2}, \frac{6}{2}\right) = \mathbf{(3, 3)}$

$M_{BC} = \left(\frac{x_B + x_C}{2}, \frac{y_B + y_C}{2}\right) = \left(\frac{5+3}{2}, \frac{4+8}{2}\right) = \left(\frac{8}{2}, \frac{12}{2}\right) = \mathbf{(4, 6)}$

$M_{AC} = \left(\frac{x_A + x_C}{2}, \frac{y_A + y_C}{2}\right) = \left(\frac{1+3}{2}, \frac{2+8}{2}\right) = \left(\frac{4}{2}, \frac{10}{2}\right) = \mathbf{(2, 5)}$

b) (0,4 pt) Determine as equações das retas:

- t: perpendicular a AB passando por $M_{AB}$

- r: perpendicular a BC passando por $M_{BC}$

- v: perpendicular a AC passando por $M_{AC}$

Gabarito

Reta t (perpendicular a AB por $M_{AB}$):

Vetor $\vec{AB} = B - A = (5-1, 4-2) = (4, 2)$. Este vetor é normal à reta $t$.

A equação de $t$ é da forma $4x + 2y + c_t = 0$.

Como $M_{AB}=(3,3)$ pertence a $t$: $4(3) + 2(3) + c_t = 0 \Rightarrow 12 + 6 + c_t = 0 \Rightarrow c_t = -18$.

$t: 4x + 2y - 18 = 0 \Rightarrow \mathbf{2x + y - 9 = 0}$.

Reta r (perpendicular a BC por $M_{BC}$):

Vetor $\vec{BC} = C - B = (3-5, 8-4) = (-2, 4)$. Este vetor é normal à reta $r$.

A equação de $r$ é da forma $-2x + 4y + c_r = 0$.

Como $M_{BC}=(4,6)$ pertence a $r$: $-2(4) + 4(6) + c_r = 0 \Rightarrow -8 + 24 + c_r = 0 \Rightarrow 16 + c_r = 0 \Rightarrow c_r = -16$.

$r: -2x + 4y - 16 = 0 \Rightarrow \mathbf{x - 2y + 8 = 0}$.

Reta v (perpendicular a AC por $M_{AC}$):

Vetor $\vec{AC} = C - A = (3-1, 8-2) = (2, 6)$. Este vetor é normal à reta $v$.

A equação de $v$ é da forma $2x + 6y + c_v = 0$.

Como $M_{AC}=(2,5)$ pertence a $v$: $2(2) + 6(5) + c_v = 0 \Rightarrow 4 + 30 + c_v = 0 \Rightarrow 34 + c_v = 0 \Rightarrow c_v = -34$.

$v: 2x + 6y - 34 = 0 \Rightarrow \mathbf{x + 3y - 17 = 0}$.

c) (0,5 pt) Calcule as intersecções: $P_1 (t \cap r)$, $P_2 (r \cap v)$, $P_3 (t \cap v)$. (Circuncentro O)

Gabarito

As três retas (mediatrizes) se interceptam no circuncentro $O$. Vamos calcular $t \cap r$.

$t: 2x + y = 9 \Rightarrow y = 9 - 2x$

$r: x - 2y = -8$

Substituindo $y$ em $r$: $x - 2(9 - 2x) = -8 \Rightarrow x - 18 + 4x = -8 \Rightarrow 5x = 10 \Rightarrow x = 2$.

$y = 9 - 2(2) = 9 - 4 = 5$.

O circuncentro é $O = (2, 5)$.

Portanto, $P_1 = P_2 = P_3 = \mathbf{(2, 5)}$.

d) (0,3 pt) Calcule o raio $R$ do círculo circunscrito.

Gabarito

O circuncentro é $O = (2, 5)$. O raio $R$ é a distância de $O$ a qualquer vértice. Usando $A=(1,2)$:

$$ R = d(O, A) = \sqrt{(x_A - x_O)^2 + (y_A - y_O)^2} $$

$$ R = \sqrt{(1 - 2)^2 + (2 - 5)^2} = \sqrt{(-1)^2 + (-3)^2} = \sqrt{1 + 9} = \sqrt{10} $$

$\mathbf{R = \sqrt{10}}$.

e) (0,5 pt) Escreva a equação da circunferência.

Gabarito

A equação da circunferência com centro $(h, k)=(2,5)$ e raio $R=\sqrt{10}$ ($R^2=10$) é:

$$ (x - h)^2 + (y - k)^2 = R^2 $$

$$ \mathbf{(x - 2)^2 + (y - 5)^2 = 10} $$

Questão 2: Distância de Ponto a Reta (2,0 pts)

Reta $r: 2x - 3y + 6 = 0$ e ponto $P = (4, 5)$.

a) (0,25 pt) Determine o vetor normal à reta $r$.

Gabarito

Para uma reta na forma $ax + by + c = 0$, o vetor normal é $\vec{n}=(a, b)$.

Para $r: 2x - 3y + 6 = 0$, o vetor normal é $\mathbf{\vec{n_r} = (2, -3)}$.

b) (0,35 pt) Encontre a equação da reta $t$ perpendicular a $r$ passando por $P$.

Gabarito

Se $t \perp r$, o vetor normal de $r$, $\vec{n_r} = (2, -3)$, é o vetor diretor de $t$, $\vec{v_t} = (2, -3)$.

Um vetor normal a $t$, $\vec{n_t}$, é perpendicular a $\vec{v_t}$. Podemos usar $\vec{n_t} = (3, 2)$.

A equação de $t$ é da forma $3x + 2y + c_t = 0$.

Como $P=(4,5)$ pertence a $t$: $3(4) + 2(5) + c_t = 0 \Rightarrow 12 + 10 + c_t = 0 \Rightarrow c_t = -22$.

$t: \mathbf{3x + 2y - 22 = 0}$.

c) (0,35 pt) Calcule $I$, intersecção entre $r$ e $t$.

Gabarito

$r: 2x - 3y = -6$

$t: 3x + 2y = 22$

Multiplicando a primeira por 2 e a segunda por 3:

$4x - 6y = -12$

$9x + 6y = 66$

Somando as equações: $13x = 54 \Rightarrow x = \frac{54}{13}$.

Substituindo $x$ em $2x - 3y = -6$: $2\left(\frac{54}{13}\right) - 3y = -6 \Rightarrow \frac{108}{13} - 3y = -\frac{78}{13}$

$-3y = -\frac{78}{13} - \frac{108}{13} = -\frac{186}{13} \Rightarrow y = \frac{186}{3 \cdot 13} = \frac{62}{13}$.

$\mathbf{I = \left(\frac{54}{13}, \frac{62}{13}\right)}$.

d) (0,25 pt) Calcule a distância entre $P$ e $I$.

Gabarito

$P=(4,5) = \left(\frac{52}{13}, \frac{65}{13}\right)$ e $I=\left(\frac{54}{13}, \frac{62}{13}\right)$.

$$ d(P, I) = \sqrt{\left(\frac{54}{13} - \frac{52}{13}\right)^2 + \left(\frac{62}{13} - \frac{65}{13}\right)^2} = \sqrt{\left(\frac{2}{13}\right)^2 + \left(\frac{-3}{13}\right)^2} $$

$$ = \sqrt{\frac{4}{169} + \frac{9}{169}} = \sqrt{\frac{13}{169}} = \frac{\sqrt{13}}{\sqrt{169}} = \mathbf{\frac{\sqrt{13}}{13}} $$

Alternativamente, usando a fórmula da distância de $P(x_0, y_0)$ à reta $ax+by+c=0$: $d = \frac{|ax_0+by_0+c|}{\sqrt{a^2+b^2}}$

$d(P,r) = \frac{|2(4) - 3(5) + 6|}{\sqrt{2^2 + (-3)^2}} = \frac{|8 - 15 + 6|}{\sqrt{4+9}} = \frac{|-1|}{\sqrt{13}} = \frac{1}{\sqrt{13}} = \mathbf{\frac{\sqrt{13}}{13}}$.

e) (0,35 pt) Escolha $A \in r$ e calcule o vetor $\vec{PA}$.

Gabarito

Reta $r: 2x - 3y + 6 = 0$. Ponto $P = (4, 5)$.

Para encontrar um ponto $A$ em $r$, fazemos $x=0$: $2(0) - 3y + 6 = 0 \Rightarrow -3y = -6 \Rightarrow y = 2$.

Então, $A = (0, 2)$ é um ponto em $r$.

Vetor $\vec{PA} = A - P = (0 - 4, 2 - 5) = \mathbf{(-4, -3)}$.

f) (0,45 pt) Calcule a projeção do vetor $\vec{PA}$ sobre o vetor normal $\vec{n_r}$.

Gabarito

$\vec{PA} = (-4, -3)$ e $\vec{n_r} = (2, -3)$.

$$ \text{proj}_{\vec{n_r}} \vec{PA} = \frac{\vec{PA} \cdot \vec{n_r}}{||\vec{n_r}||^2} \vec{n_r} $$

$\vec{PA} \cdot \vec{n_r} = (-4)(2) + (-3)(-3) = -8 + 9 = 1$.

$||\vec{n_r}||^2 = 2^2 + (-3)^2 = 4 + 9 = 13$.

$\text{proj}_{\vec{n_r}} \vec{PA} = \frac{1}{13} (2, -3) = \mathbf{\left(\frac{2}{13}, -\frac{3}{13}\right)}$.

O módulo desta projeção é $||\text{proj}_{\vec{n_r}} \vec{PA}|| = \frac{|\vec{PA} \cdot \vec{n_r}|}{||\vec{n_r}||} = \frac{|1|}{\sqrt{13}} = \frac{1}{\sqrt{13}} = \frac{\sqrt{13}}{13}$, que é a distância $d(P,r)$.

Questão 3: Inequações e Região Triangular (2,0 pts)

Pontos $A = (-2, 1)$, $B = (4, 2)$ e $C = (1, 5)$.

a) (0,5 pt) Determine as equações das retas:

- $r_1$: passa por A e B

- $r_2$: passa por B e C

- $r_3$: passa por C e A

Gabarito

$r_1$ (passa por $A=(-2,1)$ e $B=(4,2)$):

Vetor diretor $\vec{v_1} = \vec{AB} = (4 - (-2), 2 - 1) = (6, 1)$.

Vetor normal $\vec{n_1} = (1, -6)$ (troca componentes de $\vec{v_1}$ e inverte sinal de uma).

Equação de $r_1: 1x - 6y + c_1 = 0$. Usando $A(-2,1)$: $(-2) - 6(1) + c_1 = 0 \Rightarrow c_1 = 8$.

$r_1: \mathbf{x - 6y + 8 = 0}$. (Normal $\vec{n_1}=(1,-6)$)

$r_2$ (passa por $B=(4,2)$ e $C=(1,5)$):

Vetor diretor $\vec{v_2} = \vec{BC} = (1 - 4, 5 - 2) = (-3, 3)$.

Vetor normal $\vec{n_2} = (3, 3)$ ou, simplificado, $(1, 1)$.

Equação de $r_2: x + y + c_2 = 0$. Usando $B(4,2)$: $(4) + (2) + c_2 = 0 \Rightarrow c_2 = -6$.

$r_2: \mathbf{x + y - 6 = 0}$. (Normal $\vec{n_2}=(1,1)$)

$r_3$ (passa por $C=(1,5)$ e $A=(-2,1)$):

Vetor diretor $\vec{v_3} = \vec{CA} = (-2 - 1, 1 - 5) = (-3, -4)$.

Vetor normal $\vec{n_3} = (4, -3)$ (ou $(-4,3)$).

Equação de $r_3: 4x - 3y + c_3 = 0$. Usando $C(1,5)$: $4(1) - 3(5) + c_3 = 0 \Rightarrow 4 - 15 + c_3 = 0 \Rightarrow c_3 = 11$.

$r_3: \mathbf{4x - 3y + 11 = 0}$. (Normal $\vec{n_3}=(4,-3)$)

b) (0,4 pt) Ilustre o triângulo com vetores normais (como calculados em (a)).

Gabarito

A ilustração mostra o triângulo ABC e os vetores normais $\vec{n_1}$, $\vec{n_2}$, $\vec{n_3}$ a cada lado, como derivados das equações em (a), originados nos pontos médios dos respectivos segmentos.

Notas sobre o SVG: A transformação `translate(0,6) scale(1,-1)` ajusta o sistema de coordenadas. Os vetores normais são desenhados a partir dos pontos médios dos lados e escalados para um comprimento visual de aproximadamente 0.5 unidades. $\vec{n_1}=(1,-6)$, $\vec{n_2}=(1,1)$, $\vec{n_3}=(4,-3)$.

c) (0,5 pt) Determine de que lado da reta está o terceiro ponto (para definir as inequações do triângulo).

Gabarito

Para cada reta $ax+by+c=0$ e seu normal $\vec{n}=(a,b)$, substituímos as coordenadas do terceiro vértice (que não pertence à reta) na expressão $L(X) = ax+by+c$. O sinal de $L(X)$ indica o semiplano.

Para $r_1: x - 6y + 8 = 0$ (normal $\vec{n_1}=(1,-6)$) e o ponto $C=(1,5)$:

$L_1(C) = (1) - 6(5) + 8 = 1 - 30 + 8 = -21$.

Como $L_1(C) < 0$ e $\vec{n_1}$ aponta para o semiplano onde $L_1(X) > 0$, $\vec{n_1}$ aponta para fora do triângulo em relação a C. A inequação para o interior é $\mathbf{x - 6y + 8 \le 0}$.

Para $r_2: x + y - 6 = 0$ (normal $\vec{n_2}=(1,1)$) e o ponto $A=(-2,1)$:

$L_2(A) = (-2) + (1) - 6 = -7$.

Como $L_2(A) < 0$ e $\vec{n_2}$ aponta para o semiplano onde $L_2(X) > 0$, $\vec{n_2}$ aponta para fora do triângulo em relação a A. A inequação para o interior é $\mathbf{x + y - 6 \le 0}$.

Para $r_3: 4x - 3y + 11 = 0$ (normal $\vec{n_3}=(4,-3)$) e o ponto $B=(4,2)$:

$L_3(B) = 4(4) - 3(2) + 11 = 16 - 6 + 11 = 21$.

Como $L_3(B) > 0$ e $\vec{n_3}$ aponta para o semiplano onde $L_3(X) > 0$, $\vec{n_3}$ aponta para dentro do triângulo em relação a B. A inequação para o interior é $\mathbf{4x - 3y + 11 \ge 0}$.

d) (0,6 pt) Verifique se $P = (1, 2)$ está dentro do triângulo.

Gabarito

Para $P=(1,2)$ estar dentro do triângulo, deve satisfazer as três inequações:

1. $x - 6y + 8 \le 0 \Rightarrow (1) - 6(2) + 8 = 1 - 12 + 8 = -3$. Como $-3 \le 0$, é Verdadeiro.

2. $x + y - 6 \le 0 \Rightarrow (1) + (2) - 6 = 3 - 6 = -3$. Como $-3 \le 0$, é Verdadeiro.

3. $4x - 3y + 11 \ge 0 \Rightarrow 4(1) - 3(2) + 11 = 4 - 6 + 11 = 9$. Como $9 \ge 0$, é Verdadeiro.

Como $P=(1,2)$ satisfaz todas as inequações, sim, o ponto P está dentro do triângulo.

Questão 4: Cálculo de Área de Triângulos (2,0 pts)

Pontos $A = (2, 1)$, $B = (6, 3)$ e $C = (4, 7)$.

a) (0,3 pt) Determine a equação da reta AB.

Gabarito

Vetor diretor $\vec{v}_{AB} = B - A = (6-2, 3-1) = (4, 2)$.

Vetor normal $\vec{n}_{AB} = (2, -4)$ ou, simplificado, $(1, -2)$.

Equação da reta AB: $x - 2y + c = 0$. Usando $A(2,1)$: $(2) - 2(1) + c = 0 \Rightarrow c = 0$.

Reta AB: $\mathbf{x - 2y = 0}$.

b) (0,4 pt) Calcule a altura $h_C$ de $C$ até AB.

Gabarito

A altura $h_C$ é a distância do ponto $C=(4,7)$ à reta AB ($x - 2y = 0$).

$$ h_C = \frac{|ax_C + by_C + c|}{\sqrt{a^2+b^2}} = \frac{|1(4) - 2(7) + 0|}{\sqrt{1^2 + (-2)^2}} = \frac{|4 - 14|}{\sqrt{1+4}} = \frac{|-10|}{\sqrt{5}} = \frac{10}{\sqrt{5}} $$

Racionalizando: $h_C = \frac{10\sqrt{5}}{5} = \mathbf{2\sqrt{5}}$.

c) (0,4 pt) Calcule a área usando $\frac{\text{base} \times \text{altura}}{2}$.

Gabarito

Base = comprimento do segmento AB = $||\vec{AB}||$. $\vec{AB} = (4,2)$.

$||\vec{AB}|| = \sqrt{4^2 + 2^2} = \sqrt{16+4} = \sqrt{20} = \sqrt{4 \cdot 5} = 2\sqrt{5}$.

Altura $h_C = 2\sqrt{5}$.

$$ \text{Área} = \frac{\text{base} \times h_C}{2} = \frac{(2\sqrt{5})(2\sqrt{5})}{2} = \frac{4 \cdot 5}{2} = \frac{20}{2} = \mathbf{10} $$

d) (0,4 pt) Calcule o módulo do determinante dos vetores $\vec{AB}$ e $\vec{AC}$.

Gabarito

$\vec{AB} = (4, 2)$.

$\vec{AC} = C - A = (4-2, 7-1) = (2, 6)$.

$$ \det(\vec{AB}, \vec{AC}) = \begin{vmatrix} 4 & 2 \\ 2 & 6 \end{vmatrix} = (4)(6) - (2)(2) = 24 - 4 = 20 $$

Módulo do determinante = $|20| = \mathbf{20}$.

e) (0,5 pt) Compare os resultados de c) e d).

Gabarito

Resultado de (c) - Área do triângulo: $10$.

Resultado de (d) - Módulo do determinante: $20$.

A área do triângulo é metade do módulo do determinante formado pelos vetores que definem dois de seus lados com origem comum:

$$ \text{Área} = \frac{1}{2} |\det(\vec{AB}, \vec{AC})| $$

$10 = \frac{1}{2} \cdot 20$.

A comparação confirma a relação: a área (10) é metade do módulo do determinante (20).

Questão 5: Classificação de Retas (2,0 pts)

a: $x = 1 + 2t, y = 3 + t$
b: $x - 2y + 5 = 0$
c: $y = \frac{1}{2}x + 1$
d: $y = -2x + 7$
e: $\frac{x-2}{3} = \frac{y-0}{-1}$

a) (0,4 pt) Converta todas para forma geral $Ax+By+C=0$.

Gabarito

Reta a: $x = 1 + 2t, y = 3 + t$. Vetor diretor $\vec{v_a} = (2,1)$. Vetor normal $\vec{n_a} = (1,-2)$.

Equação: $x - 2y + k_a = 0$. Ponto $(1,3)$ (para $t=0$) está na reta: $1 - 2(3) + k_a = 0 \Rightarrow 1-6+k_a=0 \Rightarrow k_a=5$.

$\mathbf{a: x - 2y + 5 = 0}$

Reta b: Já está na forma geral.

$\mathbf{b: x - 2y + 5 = 0}$

Reta c: $y = \frac{1}{2}x + 1 \Rightarrow \frac{1}{2}x - y + 1 = 0$. Multiplicando por 2:

$\mathbf{c: x - 2y + 2 = 0}$

Reta d: $y = -2x + 7 \Rightarrow 2x + y - 7 = 0$.

$\mathbf{d: 2x + y - 7 = 0}$

Reta e: $\frac{x-2}{3} = \frac{y}{-1}$. Vetor diretor $\vec{v_e} = (3,-1)$. Vetor normal $\vec{n_e} = (1,3)$.

Equação: $x + 3y + k_e = 0$. Ponto $(2,0)$ está na reta: $2 + 3(0) + k_e = 0 \Rightarrow k_e=-2$.

$\mathbf{e: x + 3y - 2 = 0}$

b) (0,3 pt) Determine os vetores normais.

Gabarito

Da forma geral $Ax+By+C=0$, $\vec{n}=(A,B)$.

$\mathbf{\vec{n_a} = (1, -2)}$

$\mathbf{\vec{n_b} = (1, -2)}$

$\mathbf{\vec{n_c} = (1, -2)}$

$\mathbf{\vec{n_d} = (2, 1)}$

$\mathbf{\vec{n_e} = (1, 3)}$

c) (0,4 pt) Identifique retas paralelas.

Gabarito

Retas são paralelas se seus vetores normais são paralelos ($\vec{n_1} = k \vec{n_2}$). Se as equações são idênticas (ou uma múltipla da outra, incluindo o termo independente), são coincidentes.

$\vec{n_a}=(1,-2)$, $\vec{n_b}=(1,-2)$, $\vec{n_c}=(1,-2)$.

- $\mathbf{a \parallel b}$: Como $x-2y+5=0$ é idêntica para ambas, $\mathbf{a \text{ e } b \text{ são coincidentes}}$.

- $\mathbf{a \parallel c}$: $\vec{n_a}=\vec{n_c}$. Equações: $x-2y+5=0$ e $x-2y+2=0$. Termos independentes diferentes. $\mathbf{a \text{ e } c \text{ são paralelas distintas}}$.

- $\mathbf{b \parallel c}$: Similarmente, $\mathbf{b \text{ e } c \text{ são paralelas distintas}}$.

d) (0,4 pt) Identifique retas perpendiculares.

Gabarito

Retas são perpendiculares se o produto escalar de seus vetores normais é zero ($\vec{n_1} \cdot \vec{n_2} = 0$).

- $\vec{n_a} \cdot \vec{n_d} = (1)(2) + (-2)(1) = 2 - 2 = 0$. Então $\mathbf{a \perp d}$.

- Como $a \parallel b$, também $\mathbf{b \perp d}$.

- Como $a \parallel c$, também $\mathbf{c \perp d}$.

Outras combinações: $\vec{n_a} \cdot \vec{n_e} = (1)(1)+(-2)(3) = 1-6 = -5 \ne 0$. $\vec{n_d} \cdot \vec{n_e} = (2)(1)+(1)(3) = 2+3 = 5 \ne 0$.

e) (0,5 pt) Encontre intersecções entre não paralelas.

Gabarito

Intersecção $a \cap d$ (ou $b \cap d$, já que $a,b$ são coincidentes):

$a: x - 2y = -5 \Rightarrow x = 2y - 5$

$d: 2x + y = 7$. Substituindo $x$: $2(2y-5)+y=7 \Rightarrow 4y-10+y=7 \Rightarrow 5y=17 \Rightarrow y=\frac{17}{5}$.

$x = 2(\frac{17}{5}) - 5 = \frac{34}{5} - \frac{25}{5} = \frac{9}{5}$. $P_{ad} = P_{bd} = \mathbf{\left(\frac{9}{5}, \frac{17}{5}\right)}$.

Intersecção $c \cap d$:

$c: x - 2y = -2 \Rightarrow x = 2y - 2$

$d: 2x + y = 7$. Substituindo $x$: $2(2y-2)+y=7 \Rightarrow 4y-4+y=7 \Rightarrow 5y=11 \Rightarrow y=\frac{11}{5}$.

$x = 2(\frac{11}{5}) - 2 = \frac{22}{5} - \frac{10}{5} = \frac{12}{5}$. $P_{cd} = \mathbf{\left(\frac{12}{5}, \frac{11}{5}\right)}$.

Intersecção $a \cap e$ (ou $b \cap e$):

$a: x - 2y = -5 \Rightarrow x = 2y - 5$

$e: x + 3y = 2$. Substituindo $x$: $(2y-5)+3y=2 \Rightarrow 5y-5=2 \Rightarrow 5y=7 \Rightarrow y=\frac{7}{5}$.

$x = 2(\frac{7}{5}) - 5 = \frac{14}{5} - \frac{25}{5} = -\frac{11}{5}$. $P_{ae} = P_{be} = \mathbf{\left(-\frac{11}{5}, \frac{7}{5}\right)}$.

Intersecção $c \cap e$:

$c: x - 2y = -2 \Rightarrow x = 2y - 2$

$e: x + 3y = 2$. Substituindo $x$: $(2y-2)+3y=2 \Rightarrow 5y-2=2 \Rightarrow 5y=4 \Rightarrow y=\frac{4}{5}$.

$x = 2(\frac{4}{5}) - 2 = \frac{8}{5} - \frac{10}{5} = -\frac{2}{5}$. $P_{ce} = \mathbf{\left(-\frac{2}{5}, \frac{4}{5}\right)}$.

Intersecção $d \cap e$:

$d: 2x + y = 7 \Rightarrow y = 7 - 2x$

$e: x + 3y = 2$. Substituindo $y$: $x + 3(7-2x)=2 \Rightarrow x+21-6x=2 \Rightarrow -5x = -19 \Rightarrow x=\frac{19}{5}$.

$y = 7 - 2(\frac{19}{5}) = \frac{35}{5} - \frac{38}{5} = -\frac{3}{5}$. $P_{de} = \mathbf{\left(\frac{19}{5}, -\frac{3}{5}\right)}$.
